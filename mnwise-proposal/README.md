# 엠앤와이즈(M&Wise) 홈페이지 리뉴얼 기획안

## 프로젝트 개요
- **클라이언트**: (주)엠앤와이즈
- **현재 사이트**: https://mnwise.com/ (PHP 기반)
- **참고 사이트**: https://www.bizppurio.com/
- **목적**: 프론트엔드 리뉴얼 + CMS(직접 수정 기능) + 영문 토글
- **패키지**: PREMIUM (12P + 영문 + 유지보수 3개월)

## 페이지 구성 (12P)

### 메인 페이지
1. **index.html** — 메인 홈 (히어로 + 서비스 개요 + 고객사 + CTA)

### 서비스 페이지 (6P)
2. **service-alimtalk.html** — 카카오 알림톡
3. **service-friendtalk.html** — 카카오 친구톡
4. **service-wiseu.html** — WiseU UMS 솔루션
5. **service-wiseu-cloud.html** — WiseU Cloud
6. **service-rcs.html** — RCS 메시징
7. **service-sms.html** — SMS/LMS/MMS

### 회사 페이지 (3P)
8. **about.html** — 회사소개 (비전, 연혁, 파트너)
9. **cases.html** — 도입사례 (케이뱅크, 폭스바겐, DB손해보험 등)
10. **news.html** — 뉴스/공지사항

### 고객 페이지 (2P)
11. **contact.html** — 문의하기 (폼 + 이메일 포워딩)
12. **faq.html** — 자주 묻는 질문

## 핵심 기능
- ✅ CMS 관리자 페이지 (코딩 없이 텍스트/이미지 수정)
- ✅ KR/EN 토글 (영문 번역)
- ✅ 반응형 (PC/태블릿/모바일)
- ✅ 문의 폼 → 이메일 포워딩
- ✅ Google Analytics 연동
- ✅ 네이버/구글 SEO 최적화
- ✅ SSL 보안

## 디자인 방향
- 비즈뿌리오 참고: 깔끔한 B2B SaaS 스타일
- 카카오 파트너 신뢰감: 카카오 옐로 + 엠앤와이즈 블루 조합
- 기업 고객 대상: 전문적이되 접근하기 쉬운 톤
