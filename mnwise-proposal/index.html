FAQ

자주 묻는 질문

알림톡과 친구톡의 차이점은 무엇인가요? +

알림톡은 정보성 메시지(배송, 결제, 예약 등)로 카카오톡 채널 친구가 아니어도 발송 가능합니다. 친구톡은 마케팅/광고성 메시지로 채널 친구에게만 발송할 수 있으며, 이미지와 버튼 등 풍부한 콘텐츠를 담을 수 있습니다.

도입까지 얼마나 걸리나요? +

기본 API 연동은 평균 3일이면 완료됩니다. 복잡한 시스템 연동이 필요한 경우에도 2주 이내에 도입 가능하며, 전담 엔지니어가 전 과정을 지원합니다.

최소 발송 건수 제한이 있나요? +

최소 발송 건수 제한은 없습니다. 소규모 스타트업부터 대기업까지, 발송량에 맞는 유연한 요금제를 제공합니다. 볼륨 디스카운트도 적용 가능합니다.

대체발송(Fallback)은 어떻게 동작하나요? +

알림톡 발송 실패 시 자동으로 SMS/LMS로 대체 발송됩니다. 설정 한 번으로 모든 메시지에 적용되며, 별도 개발 없이 대시보드에서 ON/OFF 할 수 있습니다.

보안 인증은 어떤 것을 받았나요? +

ISMS(정보보호관리체계) 인증을 보유하고 있으며, 개인정보보호법과 정보통신망법을 준수합니다. 모든 데이터는 AES-256 암호화되며, 전용 IDC에서 운영됩니다.

WiseU 챗봇은 어떤 기능을 지원하나요? +

자연어 처리 기반 자동 응답, 시나리오 기반 대화 흐름, 상담사 전환, 대화 이력 관리, 통계 분석 등을 지원합니다. 별도의 개발 없이 웹 에디터에서 챗봇을 구성할 수 있습니다.
