## 공통 약관 블록 (모든 상품 상세설명 하단 의무 삽입)

```markdown
### 환불 / 보증 정책 (2025.04.22 크몽 의무 명시)
- 작업 착수 전 100% 환불
- 작업 진행 중 50% 환불 (협의 가능)
- 결과물 인도 후 결함 발생 시 무상 재작업 7일 이내
- 모든 거래는 크몽 안전결제로만 진행하며, 외부 직거래·SNS 연락은 절대 응하지 않습니다.

### 작업 프로세스 (7단계)
1. 견적 문의 / 무료 30분 화상 미팅
2. 요구사항 정리 + 제안서 송부
3. 결제 + 작업 착수
4. 1차 데모 (Standard 50% / Deluxe·Premium 30~70%)
5. 피드백 반영 + 수정
6. 최종 인도 + 사용법 가이드 (영상/문서)
7. A/S 지원 (패키지별 1~30일)

### 작업 가능 / 불가 명시
**가능**: 공개 정보 수집, 본인 운영 스토어/계정 자동화, 사내 비공개 문서 처리, B2B 워크플로우, 메신저 알림, RAG/GPT 답변 봇, API 연동
**불가**: 게임 매크로, SNS 계정 자동가입/팔로우, 광고 클릭 봇, 자동매매(자본시장법), 티켓팅, 광고 트래픽, 타사 약관 위반 행위
```
