## FAQ (12개)

**Q1. 정말 PDF 1개만으로 챗봇을 만들 수 있나요?**
A. **네**, 회사소개서·FAQ·카탈로그 등 PDF 1개만 있으면 3일 이내에 완성됩니다.

**Q2. AI가 엉뚱한 답변(환각)을 하면 어떡하나요?**
A. 문서에 없는 내용은 **"해당 정보가 없습니다"**라고 안내하도록 설정합니다. 출처 표시로 검증 가능합니다.

**Q3. 어떤 홈페이지에든 설치 가능한가요?**
A. HTML 코드 1줄 추가가 가능한 사이트면 모두 가능합니다 (**카페24, 워드프레스, Wix, 아임웹, 일반 HTML** 등).

**Q4. 월 운영 비용이 얼마나 드나요?**
A. Claude API 사용량 기준 **월 1~3만 원** 수준입니다 (하루 100건 문의 기준). 사전에 예상 비용을 안내합니다.

**Q5. 나중에 문서를 추가/변경할 수 있나요?**
A. Deluxe 이상에서 간단한 업로드로 추가 학습 가능합니다. Standard는 수정 1회 포함입니다.

**Q6. 카카오톡에서도 사용 가능한가요?**
A. **Premium에 포함**, Standard·Deluxe는 +50,000 Add-on으로 추가 가능합니다.

**Q7. 답변 톤을 우리 회사 스타일로 바꿀 수 있나요?**
A. 네, 프롬프트에서 **존댓말/친근/전문/밝은 톤** 등 맞춤 설정합니다.

**Q8. 영어/중국어/일본어 문의도 처리되나요?**
A. Claude는 다국어 우수합니다. **다국어 Add-on(+50,000)** 추가 시 최적화 답변을 제공합니다.

**Q9. 하루에 몇 건까지 처리 가능한가요?**
A. **제한 없습니다**. API 호출량에 따라 비용만 변동됩니다.

**Q10. 기존 채널톡/카카오 상담과 병행 가능한가요?**
A. 네, 별도 위젯으로 동작하므로 **기존 서비스와 독립적으로 운영**됩니다.

**Q11. 소스코드를 받을 수 있나요?**
A. 별도 상담으로 소스코드 양도 가능합니다.

**Q12. 환불이 가능한가요?**
A. 작업 착수 전 100% 환불, 착수 후에는 크몽 환불 정책을 따릅니다.
