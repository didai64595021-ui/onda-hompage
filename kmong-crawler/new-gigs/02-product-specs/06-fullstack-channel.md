## FAQ (12개)

**Q1. Standard 990,000원부터 시작인데, 무엇이 포함되나요?**
A. 채널 2개(웹+카톡 또는 슬랙) + 문서 100개 + URL 50개 + 노션 100페이지 + 자동화 3개 + 관리자 페이지 + 14일 유지보수 + 1시간 교육이 모두 포함됩니다. **무료 1.5시간 진단**으로 정확한 견적부터 받아보세요.

**Q2. 카카오 채널 사업자 인증이 안 되어 있는데요.**
A. 카카오 i 챗봇빌더 사용을 위해 사업자 인증이 필요합니다. 인증 절차도 함께 도와드립니다 (1~3일 소요). 미인증 상태에서는 슬랙·웹 위젯 + 일반 카카오 알림톡으로 대체 가능.

**Q3. 사내 ERP/CRM과 연동 가능한가요?**
A. **REST API가 있는 경우**(HubSpot, Pipedrive, 자체 ERP) 가능. API 미공개 시 Playwright RPA 또는 DB 직접 연결 협의. Standard·Deluxe는 +500,000 Add-on, Premium은 1개 기본 포함.

**Q4. 사내 데이터 보안이 가장 걱정입니다.**
A. **NDA 의무 체결**, Premium은 **자체 서버(on-prem) 호스팅** 옵션 → 클라우드 외부 노출 0. **개인정보 자동 마스킹** 기본. **침투 테스트** 옵션 가능. 작업 후 백업 즉시 삭제.

**Q5. API 비용은 얼마나 드나요?**
A. 100명 직원·일 1,000건 응답 기준 월 ₩100,000~300,000 수준 (Claude 3.5 Sonnet). 견적 시 정확한 예측치 + 비용 절감 옵션(Haiku 폴백) 함께 제안.

**Q6. 30~40일 납기가 너무 깁니다. 줄일 수 있나요?**
A. **긴급 작업** Add-on (+30~50%)으로 14일~21일까지 단축 가능. 단, 검증 단계는 압축할 수 없습니다 (환각 0% 보장 의무).

**Q7. 인도 후 직원이 시스템을 못 다루면?**
A. **임직원 화상 교육 1.5~2시간**(Premium 30명까지) + **가이드 PDF + 영상 5편** + 30~60일 무상 유지보수 + 슬랙 명령어 통합으로 사용 진입 장벽 최소화.

**Q8. 음성 응답(전화 ARS)도 가능한가요?**
A. **Premium 기본 포함**, Standard·Deluxe는 +200,000 Add-on. Whisper(STT) + 한국어 TTS로 전화/음성 챗봇 가능.

**Q9. 다국어가 필요합니다 (영어/중국어).**
A. **Premium 기본 포함**, Standard·Deluxe는 +300,000 Add-on. Claude 3.5는 영/중/일 정확도 90%+.

**Q10. 운영 중 문제가 생기면 어떻게 대응하나요?**
A. **24시간 텔레그램·슬랙 알림** + 자동 재시작 + 14~60일 무상 유지보수 + 월간 분석 리포트(Premium 3개월 무상).

**Q11. 환불 정책이 어떻게 되나요?**
A. 작업 착수 전 100% / 진행 중 30% (Premium은 작업 비중이 큼) / 인도 후 결함 7일 무상 재작업. **검증 100건 정확도 95% 미달 시 무상 재학습 1회.**

**Q12. 무료 진단은 정말 무료인가요? 영업이 부담스럽습니다.**
A. **100% 무료, 영업 강요 없음.** 화상으로 1.5시간 자동화 가능 영역만 함께 진단하고, 견적은 메일로 받으시면 됩니다. 결정 보류 시 자료만 제공해드립니다.
