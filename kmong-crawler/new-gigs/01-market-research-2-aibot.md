## 3. 최종 추천: A. RAG 사내문서 챗봇

### 추천 사유
1. **단가 가장 높음** (5.5만~165만) — 다른 후보 평균 3만대 vs 30만~110만대 → 매출 잠재력 5~30배. PREMIUM 1건 = 홈페이지 1건 매출
2. **Claude Code 단기 양산 가능** — 스택: Anthropic SDK + 임베딩 + Chroma/Supabase pgvector + Streamlit/Next.js. Max 20x 요금제로 자체 데모 무제한
3. **정책 안전 + 신규 카테고리 진입 기회** — `category/667 맞춤형 챗봇·GPT`는 2024 신설, 상위 리뷰 얇음, 1~2개월 내 페이지 1면 진입 가능. 홈페이지 5종과 동일 고객군 = 크로스셀

### 상품 3개 발전 방향
- **저가 진입 (5.5만~9.9만)**: 단순 FAQ봇 (PDF 1개, 50문항, 위젯)
- **중가 (29만~49만)**: 멀티 데이터 RAG (PDF 10개 + URL + 1차 튜닝, 관리자 페이지)
- **프리미엄 (99만~165만)**: 풀스택 (Notion/Slack/카톡 + 권한 + 대시보드 + 1개월 유지보수)

### 타겟
학원·병원·법무사·세무사 (FAQ), 쇼핑몰(상품/반품), 스타트업(사내 위키), 부동산·자동차딜러, 컨설팅·교육

### 시너지
- 매크로 백엔드 (Claude API/프롬프트/서버) 재사용
- 번들: "사내 자동화 + RAG 챗봇" = 200만+
- 데이터 흐름 통일: 매크로 수집 → RAG 학습
- 자동화 영역 리뷰를 챗봇 영역으로 이전
