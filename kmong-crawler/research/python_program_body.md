## 📌 메인 본문 (#DESCRIPTION)

### 매일 30분~2시간씩 잡아먹는 반복 업무, 파이썬 프로그램으로 자동화해드립니다

엑셀 1,000줄 정리, 사이트에서 데이터 모으기, 매일 같은 보고서 만들기 — 반복 업무에 시간 빼앗기고 계시죠. 사람 한 명 더 뽑기엔 부담스럽고, 외주 SaaS 월 구독은 매년 30~50만원이 누적되는데 결과는 비슷합니다.

파이썬 프로그램 한 번 만들어두시면, 사장님 PC 클릭 한 번으로 평소 1~2시간 걸리던 작업이 30초 안에 끝납니다. 한 번 결제로 사장님 자산입니다. 월 구독료 0원.

[검증된 자격]
✓ 자동화·크롤러·데이터 파이프라인 5,000건+ 운영
✓ 엑셀/구글시트/DB/API/웹사이트 모든 데이터 소스 대응
✓ B2B 도입 사례 12개사 (병원·학원·헬스장·부동산·쇼핑몰·공방)
✓ 결과물은 사장님 PC에 그대로 인계 — 폐업·종속 리스크 0

[Before vs After]
엑셀 정리 1,000줄: 1.5시간 수작업 → 30초 자동
사이트 데이터 모으기: 매일 1시간 복붙 → 매일 자동 발송
보고서 작성: 주 5시간 → 매일 자동 카톡/이메일
연간 외주 SaaS 비용: 30~50만 누적 → 1회 결제로 종료

[차별화 4가지]
✅ 노코드 운영 — 사장님은 클릭 한 번 또는 자동 실행
✅ 결과물 어디로든 — 엑셀/카톡/슬랙/이메일/구글시트/PDF 모두 가능
✅ 사장님 PC 또는 클라우드 — 24시간 자동 실행 옵션
✅ 인계 후 사용설명서 PDF + 무상 유지보수 (패키지별)

[적용 사례]
• 매일 아침 9시: 어제 매출/예약/문의 자동 집계 → 사장님 카톡 발송
• 1시간마다: 경쟁사 사이트 가격/재고 자동 추적 → 변동 시 알림
• 결제 발생 시: 자동 영수증 PDF 생성 → 손님 이메일 발송
• 매주 월요일: 직원 근태 엑셀 정리 → 급여 시트 자동 업데이트

[작업 흐름]
인콰이어리 → 시나리오 정의 미팅(30분 무료) → 1~2주 내 완성 → 사장님 PC 인계 → 무상 유지보수 시작

[패키지 선택 가이드]
STANDARD는 "엑셀 정리 한 가지만 자동화" 같은 단발 작업. DELUXE는 "여러 시나리오 묶어서 매일 자동 실행" (가장 많이 선택). PREMIUM은 "관리자 대시보드 + 클라우드 24시간 + 6개월 유지" 운영형.

[지금 인콰이어리 주시면 답변에 포함될 정보]
사장님 업무를 4가지 항목으로 정리만 해주시면 됩니다 (아래 템플릿).
24시간 내 1) 비슷한 사례 분석 2) 비용 절감 추천안 3) 작업 일정까지 한 번에 회신드립니다.
시나리오 정의 미팅도 무료. DELUXE 이상 결제 시 카톡 알림 연동은 추가비용 없습니다.

[인콰이어리 정보 수집 템플릿]
1. 자동화하고 싶은 업무 (예: 엑셀 정리 / 사이트 데이터 수집 / 보고서 자동 발송)
2. 데이터 소스 (엑셀 / 구글시트 / 사이트 / DB / API)
3. 결과물 형태 (엑셀 / 카톡 / 슬랙 / 이메일 / PDF / 대시보드)
4. 실행 방식 (사장님 PC 클릭 / 매일 자동 / 클라우드 24시간)

## 📌 작업 절차 (#DESCRIPTION_PROGRESS)

1단계: 인콰이어리 접수 후 24시간 내 견적서 + 작업 일정 회신
2단계: 시나리오 정의 미팅 30분 (카톡 또는 화상, 무료) — 자동화 대상·결과물·실행 방식 확정
3단계: 데이터 소스 접근 권한 확인 (구글시트 공유 / API 키 / DB 접속 정보 등)
4단계: 핵심 로직 구현 + 1차 데모 (사장님 본인 데이터로 5건 시범 실행)
5단계: 피드백 → 톤·문구·결과 형식 보강 + 예외 처리 (오류·재시도)
6단계: 사장님 PC 또는 클라우드에 인계 + PDF 사용설명서 + 무상 유지보수 시작
각 단계 완료마다 카톡으로 진행 보고드립니다.

## 📌 의뢰 전 준비사항 (#DESCRIPTION_PREPARATION)

• 자동화하고 싶은 업무 한 줄 설명 (예: "매일 9시에 어제 매출 카톡으로")
• 현재 데이터 어디에 있는지 (엑셀 파일 / 구글시트 URL / 사이트 주소 등)
• 결과물을 어디로 받고 싶으신지 (카톡 / 이메일 / 엑셀 파일 / 슬랙)
• 실행 환경 — 사장님 PC가 항상 켜져 있나요, 매일 자동이면 클라우드 추천
• 참고할 만한 다른 자동화 사례가 있으면 링크나 캡처 공유 부탁드립니다
※ 정리 어려우시면 미팅에서 함께 정리해드립니다. 그냥 오셔도 됩니다 :)

## 📌 크몽 FAQ 탭 (#FAQ — 행정/환불/계산서)

**Q. 작업 중간 취소가 가능한가요**
크몽 정책에 따라 진행률 비례 환불 가능합니다. 시나리오 미팅 전 100% / 1차 데모 전 50% / 데모 후 20%.

**Q. 세금계산서 발행되나요**
사업자 회원 결제 시 자동 발행. 개인 → 사업자 전환 처리도 30일 이내 가능.

**Q. 작업 완료 후 추가 수정 비용은**
무상 유지보수 기간 내: 시스템 오류·외부 사이트 구조 변경 대응·미세조정 무료. 신규 시나리오 추가·외부 API 신규 연동은 별도 견적.

**Q. 결과물 소스코드 공개되나요**
사장님 PC 또는 사장님 클라우드 계정에 직접 설치 + 소스 그대로 인계. 저희 서버 의존 없음, 폐업·종속 리스크 0.
