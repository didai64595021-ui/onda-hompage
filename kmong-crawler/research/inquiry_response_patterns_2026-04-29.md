## 사례 1 — 파워링크 랜딩페이지 (클린파트너, 헬스장/사우나/기업대학)

### 첫 답변 = 정보 수집 템플릿
```
견적문의 버튼+팝업 기본 포함이라 추가비용 없습니다.
기획서 없으셔도 업종에 맞게 저희가 기획부터 해드리고,
파워링크 랜딩페이지로 전환율 높게 구성해드리겠습니다 :)

맞춤 견적을 위해 아래만 확인 부탁드립니다!

[기본 정보]
1. 업체명 / 대표자명 / 전화번호
2. 서비스 지역 (ex. 서울 전 지역, 수도권 등)
3. 주요 서비스 항목 (대표 서비스 3~5개)
4. 로고나 현장사진 (없으셔도 업종에 맞게 제작합니다)

[홈페이지 구성]
5. 한 페이지에 다 담을까요, 메뉴별로 나눌까요?
   - 한 페이지로 깔끔하게 (소개~문의 한 번에)
   - 메뉴별 페이지 분리 (소개/서비스/문의 등 나눠서)
6. 납품 후 사진/문구 직접 바꿀 일이 있을까요?
   - 네, 직접 바꾸고 싶어요
   - 아니요, 수정 요청 드릴게요
7. 추가 기능 (선택):
   - 카카오톡 채널 연동 / 네이버 예약 / 인스타그램 피드
   - 상단 공지·이벤트 배너 / QR 코드 / 특별히 없음

확인 주시면 맞춤 견적 안내드리겠습니다!
```

### 답변 후 견적 산정
- 메뉴 2~3개 분리 → 20만원
- 메뉴 4개 이상 + 유지보수 1개월 → 35만원
- 3개 메뉴 + 서브 1개 → DELUXE + 추가 = **22만원** (조립식)

### 패턴 키워드
- "추가비용 없습니다" 무료 옵션 강조
- ":)" 친근 이모지
- 단계별 옵션 가시화 ("이렇게 vs 이렇게")
- 사장님이 직접 답변할 항목 vs 전부 알아서 처리할 항목 분리

## 사례 2 — 만정누수탐지 (사이트 분석형 추천)

### 사이트 분석 후 추천안
```
만정누수탐지 사이트 확인해봤는데 서브페이지가 7개 이상인 사이트입니다.

동일하게 만들면 비용이 꽤 올라가서, 핵심 페이지만 추려서 진행하는 걸 추천드립니다.

• 메인 페이지: 회사소개 + 시공과정 + 연락처
• 서비스 페이지: 누수탐지/방수공사 등 서비스 안내
• 블로그, 카카오톡은 버튼으로 바로 연결

이렇게 2페이지 구성이면 DELUXE 패키지(20만원)로 진행 가능합니다.
```

### 추가 옵션 — 단가 + 업종 추천
```
• +3만: 카카오톡 채널 상담 연동
• +3만: QR 코드 제작 (오프라인 현장에서 홈페이지 유입용)
• +5만: CMS 관리 기능 (납품 후 직접 수정 가능)

개인적으로 누수탐지 업종이시면 CMS는 추천드립니다.
시공사례 사진이나 서비스 지역 추가할 때 저희 거치지 않고 직접 수정 가능.
```

### 호스팅·도메인 안내
```
호스팅/도메인은 도메인 이미 구매하셨으니 호스팅만 연결하면 되고,
월 500~5,000원 수준입니다. 세팅은 저희가 전부 처리해드립니다.
```

→ **결제 전환됨**: "카카오톡 채널 + CMS = 28만원으로 DELUXE 진행하겠습니다"

### 패턴 키워드
- 고객 사이트 직접 확인 (디테일)
- "비용 절감 추천" (셀러 → 사장님 입장 대변)
- 옵션을 +금액 라인으로 가독성
- 업종별 맞춤 옵션 추천 (현장형 = QR, 시공형 = CMS)
- "저희가 전부 처리해드립니다" 부담 제로 강조

## 🎯 핵심 패턴 8가지 (신상품 본문/auto-reply 반영용)

1. **첫 답변 = 정보 수집 템플릿**: 기본정보 + 구성 + CMS여부 + 추가옵션, 4섹션
2. **무료 옵션 명확히 강조**: "추가비용 없습니다", "기본 포함"
3. **단계별 옵션 가시화**: "메뉴 2~3개 vs 4개 이상", 명확한 가격 분기점
4. **업종 분석 후 맞춤 추천**: 사장님 사이트/업종 보고 비용 절감 안 제시 (셀러 입장 X, 사장님 입장 ✓)
5. **+금액 옵션 라인**: "+3만 카톡연동 / +5만 CMS" 한 줄당 한 옵션
6. **호스팅·도메인 안내 표준 문구**: "월 500~5,000원, 저희가 전부 처리"
7. **친근 톤 + 이모지 ":)"**: 전문성 보존하면서 부담 제로
8. **체크-인 클로징**: "확인 주시면 맞춤 견적 안내드리겠습니다"

## 🆕 신상품 적용 매핑 (Python 프로그램 / 앱 출시)

### Python 프로그램 신상품 — 첫 답변 정보 수집 템플릿 (사례 1 응용)
```
파이썬 프로그램 제작 문의 감사합니다 :)
업종/요구에 맞춰 기획부터 제작까지 진행하니 기획서 없으셔도 됩니다.

맞춤 견적을 위해 아래만 확인 부탁드립니다!

[기본 정보]
1. 어떤 업무·반복작업을 자동화하고 싶으신가요? (예: 엑셀 가공/사이트 데이터 수집/보고서 생성/스케줄 발송)
2. 어떤 데이터를 다루시나요? (엑셀/구글시트/웹사이트/DB/API)
3. 결과물은 어떤 형태로 받고 싶으신가요? (엑셀 파일/카톡 알림/슬랙/이메일/대시보드)
4. 실행 환경 (사장님 PC 수동 실행 / 매일 자동 실행 / 클라우드 24시간)

[추가 기능]
- 카카오톡 알림 연동
- 구글시트 자동 업데이트
- 결과물 PDF/엑셀 자동 발송
- 관리자 대시보드 (실행 로그·재시도)
- 특별히 없음

확인 주시면 1~2일 내 맞춤 견적 + 작업일정 안내드리겠습니다!
```

### 앱 출시 신상품 — 첫 답변 (사례 2 응용, 시장 분석형)
```
앱 출시 문의 감사합니다 :)

업종/기능 듣고 비슷한 앱 시장에서 찾아본 후, 핵심 기능만 추려서 비용 절감 안으로 추천드릴 수 있습니다.

맞춤 견적을 위해 아래만 확인 부탁드립니다!

[기본 정보]
1. 어떤 앱을 만들고 싶으신가요? (한 줄 설명)
2. 참고 앱이 있으시면 이름이나 스토어 링크 부탁드립니다
3. 안드로이드만 / 아이폰만 / 둘 다? (비용 다릅니다)
4. 출시 시점 (1개월 / 2~3개월 / 여유)

[추가 기능 — 자주 선택하시는 것]
- 회원가입·로그인 (카카오/구글/애플 소셜로그인 포함)
- 결제 (카드/토스/카카오페이)
- 푸시 알림
- 채팅·1:1 문의
- 관리자 페이지 (사용자/콘텐츠 관리)

[출시 대행 — 별도 옵션]
- 안드로이드 (Play Store) 등록 대행: +5만
- 아이폰 (App Store) 등록 대행: +10만 (애플 개발자 계정 별도)

확인 주시면 비슷한 앱 시장 분석 + 견적 + 작업일정 안내드리겠습니다!
```

## 본문 7단계에 반영할 패턴

기존 KMONG_CONTEXT.md PHASE 2 7단계 (후킹→자격→B/A→프로세스→패키지→FAQ→CTA)에 **인콰이어리 정보 수집 템플릿을 마지막 CTA 직전에 삽입**:

```
[지금 인콰이어리 주시면 답변에 포함될 정보]
- 업종·요구사항 4개 항목으로 정리만 해주시면 됩니다 (위 템플릿)
- 24시간 내 1) 비슷한 사례 분석 2) 비용 절감 추천안 3) 작업 일정까지 한 번에 회신드립니다
- 견적문의·전화걸기 팝업·카톡채널 연동은 무료 기본 포함
```

이 표현이 들어가면 인콰이어리 진입 부담을 낮춤 (무엇을 말해야 할지 모르는 사장님이 가장 큰 이탈 요인).

## auto-reply 봇 적용 우선순위

1. **`lib/reply-generator.js`** — 첫 답변 시 위 정보수집 템플릿을 카테고리별로 자동 삽입
2. **`lib/customer-profile.js`** — 사장님 카테고리(헬스장/공사/누수 등) 추출 시 옵션 추천 다르게
3. **`lib/converted-examples.js`** — 본 사례 2건을 시드 학습 데이터로 추가 (이미 결제 전환 패턴)
