
## 네이버ID 빈칸 원인 분석 (19:36)
- BaseNaverBlog:xxx 키는 사업자가 플레이스에 공식 블로그 연결했을 때만 생성
- 빈칸 = 블로그 없음 or 블로그는 있지만 플레이스 미연결 or 인스타/홈페이지만 운영
- **추가 추출 방법 (확립):**
  1. BaseNaverBlog 키 (플레이스 HTML)
  2. blogId 키 (플레이스 HTML)
  3. blog.naver.com URL 패턴 (플레이스 HTML)
  4. 홈페이지 URL 역추적 → 홈페이지 내 블로그 링크 추출
  5. 네이버 검색 "업체명 site:blog.naver.com" → title 매칭
  6. 네이버 검색 "업체명 블로그" → blog.naver.com URL 추출
  7. 인스타그램 폴백 (@계정)
