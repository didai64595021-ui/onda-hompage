## 크몽 문의 응대 전환율 규칙 (영구)

### 핵심 원칙
- 질문 3~4개 이내 (필수만 먼저, 나머지 작업 중 확인)
- 메시지 10줄 이내 (사장님은 바쁘다)
- 가격 확정 상품이면 "견적 안내" 하지 말고 포함이라고 단언
- 호스팅/기술 용어 의뢰인에게 묻지 말고 우리가 알아서 처리
- "편하게 보내주세요" → "주시면 바로 착수합니다" (행동 유도)
- 이모지 금지 → :) 등 텍스트 이모티콘 사용

1. 업체명 / 대표자명 / 전화번호
2. 서비스 지역
3. 도메인 주소
4. 로고/현장사진 (없어도 괜찮다고 안내)

- 소개글/문구
- 사업자등록번호

- 주소/이메일
