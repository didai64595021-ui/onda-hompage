## 관리자 지시사항 (중요 - 영구 적용)

### 1. 스텝별 자동 보고 (모든 프로젝트)
- 작업 시작 → "🔨 작업 시작: [내용]"
- 스텝 완료 → "✅ [스텝명] 완료: [결과]"
- 오류 발생 → "⚠️ 오류: [내용] → 복구 중..."
- 복구 완료 → "🔄 복구 완료, 재진행"
- 배포 완료 → "🚀 배포 완료: [링크]"
- **한다고 한 건 끝나면 반드시 결과 보고 (재촉 안 해도)**

### 2. 오류 시 자동 이어서 진행 (모든 프로젝트 봇)
- 오류로 멈추면 **직전 작업 내용 기억해서 알아서 이어서 진행**
- 멈춘 채로 방치 금지 → 자동 복구 후 재개
- 보고는 당연히 하되, 보고만 하고 멈추지 말 것

### 3. 작업 완료 후 테스트+디버깅 무한 반복 (모든 프로젝트 공통)
- 작업 종료 후 반드시 **빌드 테스트 → 오류 수정 → 재테스트** 반복
- 오류 0개가 될 때까지 계속 반복
- 테스트 통과해야만 "완료" 보고
- 이 규칙은 모든 프로젝트, 모든 작업에 적용

### 4. 오류/디버깅 능동 해결
- 오류 발생 시 관리자에게 물어보지 말고 **스스로 최선의 방법 찾아서 해결**
- Claude Code API 안 되면 직접 코딩
- API 서버 장애면 대안 찾기 (작업 쪼개기, 직접 수정 등)
- 항상 능동적으로 디버깅
- **"안 된다"고 보고만 하지 말고, 해결 후 보고**

### 3. 모든 프로젝트 자동 보고 시스템
- 모든 프로젝트에 스텝별 상황 보고 적용
- 신규 프로젝트에도 동일 적용

### 4. API 키 충돌 해결법
- ANTHROPIC_API_KEY 환경변수가 Claude Code OAuth 인증과 충돌
- Claude Code 실행 시 `ANTHROPIC_API_KEY=""` 로 무효화 필요
