### crawler_engine.py 주요 변경 (2977줄 → 정상 기준)
1. **Selenium → GraphQL 방식**: iframe 진입 → 2페이지 클릭 → Performance 로그에서 원본 쿼리 캡처 → JS fetch로 페이지 1~N 호출
2. **Promise.all 병렬 호출**: 순차 10초 → 병렬 1초 (10배 속도 향상)
3. **base64 인코딩**: JS 문자열 이스케이프 문제 회피 (`atob()` 사용)
4. **마지막 페이지 display 보충**: 0건 시 display 줄여서 재시도 (70→50→30→20→10)
5. **1페이지 키워드 HTML 폴백**: 2페이지 없으면 HTML에서 같은 JSON 객체의 id+name 추출
6. **Performance 로그 캡처**: `opts.set_capability("goog:loggingPrefs", {"performance": "ALL"})`
