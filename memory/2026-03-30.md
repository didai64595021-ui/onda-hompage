- 3개 탭: ① CPC 소재테스트, ② 전환 퍼널, ③ 서비스별 문의량

### DB 테이블 (Supabase, 생성 완료)
- kmong_products: 14개 상품 마스터
- kmong_cpc_daily: CPC 일별 데이터 (노출/클릭/비용/CTR)
- kmong_creative_changes: 소재 변경 이력
- kmong_inquiries: 문의 데이터 (상태: pending/quoted/paid/cancelled)
- kmong_orders: 주문 데이터
- 샘플 데이터 15건 CPC + 8건 문의 + 3건 주문 + 3건 소재변경 입력 완료

### 대시보드 기능
- 탭1: KPI 카드 4개 + 일별 추이 라인차트 + CTR비교 바차트 + CPC비용 차트 + 성과테이블 + 소재변경이력
- 탭2: 5단계 퍼널 시각화 + 요일×시간 히트맵 + 전환율 테이블 + 전환 소요일 차트
- 탭3: 카테고리별 도넛 + 월별 추이 + 문의vs결제 비교 + 최근 문의 목록
- 공통: 데이터 입력폼(CPC/문의/주문/소재변경) + CSV 업로드 + 날짜범위 필터 + 테이블 정렬 + 반응형
