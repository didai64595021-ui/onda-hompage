## onda-youtube-investment 업로드 디버깅 (03-02 오후)

### YouTube 업로드 실패 원인 3가지
1. **CH03 (모르모트)**: `uploadLimitExceeded` — 일일 업로드 할당량 초과
2. **CH01/CH02**: `invalid_grant` — OAuth 토큰 만료, localhost redirect라 원격 갱신 불가
3. **CH04-CH08**: 토큰 미설정

### 구현 완료
- `uploadWithChannelRotation()` — ch-3→ch-1→ch-2→ch-4~8 순서 자동 시도
- 3x retry + exponential backoff (5s/15s/45s) + 매 시도 OAuth 재갱신
- `--recover` 모드 — DB에서 미완료 작업 스캔 후 마지막 단계부터 재개
- `scripts/oauth-refresh.ts` — OAuth 토큰 재발급 2단계 (URL 생성 → 코드 교환)
- `scripts/upload-now.ts` — R2에서 직접 YouTube 업로드 + 채널 자동 로테이션
- `scripts/test-channels.ts` — 전 채널 OAuth/업로드 가용성 테스트

### 차단 상태
- CH01/CH02 토큰 갱신 → 관리자가 브라우저에서 OAuth 인증 필요 (redirect_uri=localhost)
- CH03 쿼터 리셋 대기 (자정 Pacific / 오전 9시 KST 예상이나 13시에도 미복구)
