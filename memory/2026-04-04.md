## 온다 애드 (Onda Ad) SaaS 기획 (18:30~19:38)

### 사업 모델 확정
- **서비스**: 네이버 검색광고 올인원 AI 관리 솔루션
- **구독료**: 무료 (0원)
- **조건**: 네이버 검색광고 대행 계약 체결
- **수익구조**: 대대행 14% 요율 (공인대행사가 15% 수령, 1% 공인대행사 몫)
- **환급 포인트**: 소진액 3%(100만 미만) / 5%(100~500만) / 7%(500만+)
- **포인트 사용처**: 홈페이지 제작, 플레이스 마케팅, 셀프마케팅 트래픽 구매, 소재 디자인 등
- **부정클릭 환불 수수료**: 무료 (0원) — 보라웨어 대비 킬러 포인트
- **홈페이지 단가**: 크몽 동일 (STANDARD 12만/DELUXE 20만/PREMIUM 35만) + 옵션 체크박스

### 핵심 기능 7가지
1. 통합 CPC 대시보드
2. 부정클릭 탐지 + 환불 (웹사이트: JS 핑거프린팅 / 플레이스: API 데이터 분석)
3. AI 소재 생성 (GPT Image + 카피)
4. CPC 단가 인하 (키워드별 맞춤 랜딩페이지 → 품질점수 최적화)
5. 입찰가 자동 최적화
6. 키워드 확장 (AI 롱테일 발굴)
7. ROAS 극대화 (종합 절감액 리포트)

### 인증 구조
- **최초 로그인**: 네이버 소셜 로그인 (필수, 유일)
- **API 키**: 고객 직접 발급 → 입력 (자동추출 기능 추후 추가)
- **대행 권한**: 별도 안내 → 광고주센터에서 권한 이전

### 대대행 구조 확인
- 온다마케팅 = 대대행 (공인대행사 밑)
- 공인대행사가 15% 수령 → 온다에 14% 지급 (1% 수수료)
- 환불 신청: 대대행 권한으로 직접 불가할 수 있음 → 증빙 자동생성 후 공인대행사 경유 또는 광고주 계정으로 접수

### 플레이스 CPC 부정클릭
- 플레이스 클릭 후 네이버 도메인으로 이동 → JS 트래킹 코드 삽입 불가
- 대안: API 데이터 기반 이상 패턴 탐지 (시간대별 클릭 급증 + 전환율 괴리)

### 기획서 파일

### 셀프마케팅 트래픽 자동주문 기능 (검토)
- onda-traffic-bot (기존 Playwright 자동 주문 봇)을 API로 래핑
- 고객이 셀프마케팅 페이지에서 매체/타수/키워드 입력 → 포인트 차감 → 자동 주문 처리
- 흐름: 주문 UI → Worker API → Supabase 저장 → traffic-bot 감지 → Playwright 자동주문 → 상태 업데이트
- 시장에 없는 유일한 기능 (고객 원클릭 트래픽 주문)
- 아직 기획 단계, 코드 미작성
