> **우리 강점**: "이미 있는 사이트에 CMS(직접 수정 기능)를 입혀드립니다"

## 상품 1: 기존 홈페이지에 CMS 추가 (핵심 차별화 상품)

### 서비스 제목
```
기존 홈페이지에 CMS 탑재 | 코딩 없이 직접 수정하는 관리 시스템 추가
```

### 카테고리
```
IT·프로그래밍 > 웹사이트 개발 > 홈페이지
```

### 가격 구성

| 패키지 | 가격 | 내용 |
|--------|------|------|
| BASIC | 200,000원 | 텍스트+이미지 수정 CMS 추가 (5개 섹션까지) |
| STANDARD | 400,000원 | 전체 CMS (텍스트/이미지/순서변경/추가삭제) + 테마 설정 패널 |
| PREMIUM | 700,000원 | 전체 CMS + 위젯(FAQ/가격표/갤러리) + SEO 설정 + 1개월 유지보수 |

### 추가 옵션
```
- 섹션 추가 (1개당): +30,000원
- 위젯 추가 (FAQ/갤러리/가격표 등 1종): +50,000원
- 반응형 보정 (미반응형 사이트 → 반응형): +200,000원
- 급행 (3일 이내): +150,000원
- 유지보수 연장 (월): +50,000원
```

### 서비스 상세 설명
```
✏️ 이미 가지고 계신 홈페이지, 수정할 때마다 업체에 연락하시나요?

━━━━━━━━━━━━━━━━━━━━━━━━

💡 이런 불편함이 있으시다면

• 텍스트 하나 바꾸려면 업체에 요청 → 2~3일 대기 → 추가 비용
• 이미지 교체할 때마다 매번 연락
• 메뉴 가격이 바뀌었는데 사이트는 옛날 가격 그대로
• 이벤트 배너 올리고 싶은데 혼자서는 못 건드림
• 새로 만들기엔 아깝고, 지금 사이트에 수정 기능만 있으면 되는데...

→ 기존 홈페이지는 그대로 두고, CMS(직접 수정 시스템)만 얹어드립니다.

━━━━━━━━━━━━━━━━━━━━━━━━

📌 CMS가 추가되면 이렇게 바뀝니다

✅ 텍스트: 화면에서 클릭 → 바로 수정 → 저장 끝
✅ 이미지: 사진 클릭 → 새 이미지 업로드 → 즉시 반영
✅ 섹션 순서: 드래그&드롭으로 위치 변경
✅ 항목 추가/삭제: 서비스 카드, FAQ, 갤러리 등 자유롭게
✅ 디자인 변경: 색상, 폰트, 카드 모양 등 설정 패널에서 조절
✅ 별도 프로그램 설치 없음: 웹 브라우저에서 바로 편집

🚫 워드프레스 설치 필요 없음
🚫 FTP 접속 필요 없음
🚫 코딩 지식 필요 없음

━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 아임웹/Wix/스퀘어스페이스 등 폐쇄형 빌더는 적용 불가

📂 작업 흐름

① 현재 사이트 URL 전달 + 수정하고 싶은 부분 정리 (1일)
② 사이트 분석 + CMS 적용 범위 안내 (1일)
③ CMS 코드 탑재 + 테스트 (2~4일)
④ 사용법 안내 (화면 녹화 영상 제공)
⑤ 최종 확인 + 납품

📌 총 작업 기간: 5~7일

💬 자주 묻는 질문

Q. 기존 디자인이 바뀌나요?
A. 아닙니다. 기존 사이트 디자인은 100% 유지하고, 수정 기능만 추가합니다.

Q. 사이트가 느려지지 않나요?
A. CMS 코드는 관리자 모드에서만 작동합니다. 일반 방문자에게는 영향 없습니다.

Q. 수정한 내용이 바로 반영되나요?
A. 네. 저장 버튼 누르면 즉시 반영됩니다.

Q. 사용법이 어렵지 않나요?
A. 화면 녹화 영상으로 사용법을 안내드립니다. 컴퓨터 기본 사용 가능하면 누구나 할 수 있습니다.

Q. 여러 명이 관리할 수 있나요?
A. 관리자 링크만 공유하면 됩니다. 별도 계정 시스템도 추가 가능합니다 (옵션).
```

A. 아닙니다. 자체 개발 경량 CMS를 사용합니다. 워드프레스보다 빠르고 가볍습니다.
