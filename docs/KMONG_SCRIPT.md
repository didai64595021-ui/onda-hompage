# 크몽 문의 응대 스크립트 (751791 상품)

## 플로우 (3단계)
1. **질문** — 고객 문의사항 답변 + 기본/견적 관련 질문
2. **견적** — 답변 기반으로 견적 산출 안내
3. **착수** — 결제 후 착수 안내

## 규칙
- 이모지 금지 → :) 텍스트 이모티콘만
- 호스팅/기술 용어 묻지 말고 알아서 처리
- 가격 확정 항목은 "포함입니다" 단언
- "편하게~" 금지 → "주시면 바로 진행하겠습니다"

---

## 1단계 — 질문 (고객 문의사항 답변 + 기본/견적 질문)

```
안녕하세요 대표님, 문의 감사합니다 :)

{고객 문의사항에 대한 답변}

홈페이지 구성을 위해 아래만 확인 부탁드립니다!

[기본 정보]
1. 업체명 / 대표자명 / 전화번호
2. 서비스 지역 (ex. 서울 전 지역, 수도권 등)
3. 주요 서비스 항목 (대표 서비스 3~5개)
4. 로고나 현장사진 (없으셔도 업종에 맞게 제작합니다)

[견적 관련]
5. 아래 추가 옵션 중 필요한 게 있으시면 알려주세요 :)
   - CMS 직접 수정 기능 (+5만)
   - 서브페이지 추가 (1개당 +2만)
   - 카카오톡 채널 연동 (+3만)
   - 네이버 예약 버튼 (+3만)
   - 상단 공지/이벤트 배너 (+3만)
   - QR 코드 제작 (+3만)
   - 인스타 피드 연동 (+5만)
   - SEO 심화 등록 (+5만)
   필요 없으시면 기본으로 진행됩니다!
```

### 고객 문의사항 답변 예시

| 고객 질문 | 답변 |
|-----------|------|
| 견적문의 버튼/팝업 추가하고 싶어요 | 견적문의 버튼+팝업 기본 포함이라 추가비용 없습니다 :) |
| 파워링크 연결용으로 쓰려고요 | 파워링크 랜딩페이지로 전환율 높게 구성해드립니다 :) |
| 기획서/디자인 없어요 | 업종에 맞는 구성으로 저희가 기획부터 해드립니다 :) |
| 직접 수정할 수 있나요? | CMS 관리 기능 추가하시면 코딩 없이 직접 수정 가능합니다 (+5만원 옵션) |
| 모바일에서도 잘 보이나요? | 모바일/PC 동시 대응 반응형으로 제작합니다 :) |
| 얼마나 걸리나요? | 정보 주시면 3일 내 완성해드립니다 :) |
| 수정 가능한가요? | 수정 무제한이고 7일 무료 AS 포함입니다 :) |

## 2단계 — 견적 산출 (고객 답변 수신 후)

```
감사합니다 대표님! 확인했습니다 :)

견적 안내드립니다.

- 기본 패키지: 12만원
  (반응형 1페이지 + 전화/카카오 버튼 + 지도 + SSL + 검색등록 + 수정무제한 + 7일AS)
{추가 옵션 선택한 경우}
- {옵션명}: +{가격}
---
합계: {총액}원

도메인/호스팅 셋팅은 별도 비용 없이 해드립니다.
결제 진행해주시면 바로 착수하겠습니다!
```

### 추가 옵션 없는 경우
```
감사합니다 대표님! 확인했습니다 :)

기본 패키지 12만원으로 진행됩니다.
(반응형 1페이지 + 전화/카카오 버튼 + 지도 + SSL + 검색등록 + 수정무제한 + 7일AS)

도메인/호스팅 셋팅 별도 비용 없이 해드립니다.
결제 진행해주시면 바로 착수하겠습니다!
```

---

## 3단계 — 착수 안내 (결제 확인 후)

```
결제 확인했습니다 대표님, 감사합니다 :)

3일 내 1차 시안 보내드리겠습니다.
진행하면서 궁금하신 점 있으시면 편하게 말씀해주세요!
```

---

## 기본 포함 사항 (12만원)
- 반응형 1페이지 (모바일/PC)
- 전화걸기 + 카카오톡 상담 버튼
- 견적문의 버튼/팝업
- 네이버/구글 지도 삽입
- SSL 보안(https) + 검색엔진 등록
- 수정 무제한 + 7일 무료 AS
- 도메인/호스팅 셋팅
