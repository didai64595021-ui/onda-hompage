# 크몽 문의 응대 스크립트

## 플로우 (3단계, 영구)
1. **질문** — 고객 문의사항 답변 + 기본정보 + 상품 선정 질문
2. **견적** — 답변 기반 상품 확정 + 견적 산출 + 필요시 추가 질문/업셀링
3. **착수** — 결제 확인 후 착수 안내

## 규칙
- 이모지 금지 → :) 텍스트 이모티콘만
- 호스팅/기술 용어 묻지 말고 알아서 처리
- 가격 확정 항목은 "포함입니다" 단언
- "편하게~" 금지 → "주시면 바로 진행하겠습니다"
- **1단계에서 "바로 착수" 절대 금지** — 질문→견적→착수 순서 필수

## 크몽 링크 수신 시 프로세스 (영구)
1. **크롤링** — Playwright로 크몽 상품 페이지 접속, 3개 패키지 탭 클릭하여 가격/구성 추출
2. **상세페이지** — web_fetch로 상세 설명 + 추가 옵션 추출
3. **데이터 검증** — 패키지 3개 가격/구성 + 추가 옵션 목록 정리 → 관리자에게 확인
4. **질문서 작성** — 검증된 데이터 기반으로 1단계 질문 스크립트 생성

### 크롤링 코드 (Playwright)
```python
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto(URL, wait_until='networkidle', timeout=30000)
    page.wait_for_timeout(3000)
    for pkg in ['STANDARD', 'DELUXE', 'PREMIUM']:
        tab = page.query_selector(f'text={pkg}')
        if tab:
            tab.click()
            page.wait_for_timeout(800)
    # 패키지 영역 + 상세 텍스트 추출
```

## 1단계 — 질문 (문의사항 답변 + 기본정보 + 상품 선정)

```
안녕하세요 대표님, 문의 감사합니다 :)

맞춤 견적을 위해 아래만 확인 부탁드립니다!

[기본 정보]
1. 업체명 / 대표자명 / 전화번호
2. 서비스 지역 (ex. 서울 전 지역, 수도권 등)
3. 주요 서비스 항목 (대표 서비스 3~5개)
4. 로고나 현장사진 (없으셔도 업종에 맞게 제작합니다)

[홈페이지 구성]
5. 한 페이지에 다 담을까요, 메뉴별로 나눌까요?
   - 한 페이지로 깔끔하게 (소개~문의 한 번에)
   - 메뉴별 페이지 분리 (소개/서비스/문의 등 나눠서)
   * 페이지 수에 따라 금액이 달라질 수 있습니다

6. 납품 후 사진이나 문구를 직접 바꿀 일이 있으실까요?
   - 네, 직접 바꾸고 싶어요
   - 아니요, 수정 요청 드릴게요

7. 추가로 넣고 싶은 기능이 있으시면 골라주세요 :)
   - 카카오톡 채널 연동
   - 네이버 예약 버튼
   - 인스타그램 피드 연동
   - 상단 공지/이벤트 배너
   - QR 코드 (매장에서 홈페이지 바로 연결)

확인 주시면 맞춤 견적 안내드리겠습니다!
```
