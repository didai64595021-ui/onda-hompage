# 크몽 문의 응대 스크립트 (751791 상품)

## 규칙
- 질문 4개 이내
- 10줄 이내
- 이모지 금지 → :) 텍스트 이모티콘만
- 호스팅/기술 용어 묻지 말고 알아서 처리
- "편하게~" 금지 → "주시면 바로 착수합니다"
- 가격 확정 항목은 "포함입니다" 단언

## 스크립트 구조

### 1단 — 인사 + 고객 문의사항 답변 + 기본 질문

```
안녕하세요 대표님, 문의 감사합니다 :)

{고객 문의사항에 대한 답변 — 아래 예시 참고}

홈페이지 제작에 필요한 정보 아래만 확인 부탁드립니다!

1. 업체명 / 대표자명 / 전화번호
2. 서비스 지역 (ex. 서울 전 지역, 수도권 등)
3. 주요 서비스 항목 (대표 서비스 3~5개)
4. 로고나 현장사진 (없으셔도 업종에 맞게 제작합니다)

도메인/호스팅 셋팅 전부 해드리니 신경 안 쓰셔도 됩니다.
주시면 바로 착수합니다!
```

### 고객 문의사항 답변 예시

**"견적문의 버튼/팝업 추가하고 싶어요"**
→ `견적문의 버튼+팝업 기본 포함이라 추가비용 없습니다 :)`

**"파워링크 연결용으로 쓰려고요"**
→ `파워링크 랜딩페이지로 전환율 높게 구성해드립니다 :)`

**"기획서/디자인 없어요"**
→ `업종에 맞는 구성으로 저희가 기획부터 해드립니다 :)`

**"카카오톡 채널 연결하고 싶어요"**
→ `카카오톡 채널 연동 가능합니다 (+3만원 옵션)`

**"직접 수정할 수 있나요?"**
→ `CMS 관리 기능 추가하시면 코딩 없이 직접 수정 가능합니다 (+5만원 옵션)`

**"페이지 여러 개 필요해요"**
→ `기본 1페이지 + 서브페이지 추가 가능합니다 (1개당 +2만원)`

**"네이버 예약 연동되나요?"**
→ `네이버 예약 버튼 연동 가능합니다 (+3만원 옵션)`

**"인스타 피드 넣고 싶어요"**
→ `인스타그램 피드 실시간 연동 가능합니다 (+5만원 옵션)`

**"SEO/검색 잘 되게 해주세요"**
→ `구글/네이버 검색 등록 기본 포함이고, 심화 SEO(메타태그+사이트맵+구조화데이터)는 +5만원 옵션입니다 :)`

**"모바일에서도 잘 보이나요?"**
→ `모바일/PC 동시 대응 반응형으로 제작하니 걱정 안 하셔도 됩니다 :)`

**"얼마나 걸리나요?"**
→ `정보 주시면 3일 내 완성해드립니다 :)`

**"수정 가능한가요?"**
→ `수정 무제한이고 7일 무료 AS 포함입니다 :)`

## 기본 포함 사항 (12만원)
- 반응형 1페이지 (모바일/PC)
- 전화걸기 + 카카오톡 상담 버튼
- 견적문의 버튼/팝업
- 네이버/구글 지도 삽입
- SSL 보안(https) + 검색엔진 등록
- 수정 무제한 + 7일 무료 AS
- 도메인/호스팅 셋팅
