# 온다 애드 (Onda Ad) — 서비스 기획서

## 1. 서비스 개요

**한줄 정의**: 네이버 검색광고 올인원 AI 관리 솔루션

**포지션**: 부정클릭 방지 + AI 소재 자동화 + CPC 절감 + 통합 대시보드를 하나로

**타겟**: 네이버 검색광고 운영 중인 자영업자 · 중소기업 · 병의원

## 2. 상품 구조 (3단계 고객 확보 퍼널)

### 플랜 A: 풀 서비스 (무료)
| 항목 | 내용 |
|------|------|
| **월 이용료** | 무료 |
| **조건** | 네이버 검색광고 대행 계약 체결 |
| **부정클릭 환불 수수료** | 무료 (0원) |
| **포함** | 대시보드 + 부정클릭 + AI 소재 + CPC 절감 + 키워드 확장 + 소재 자동 교체 + 롱테일 자동 등록 + 포인트 환급 |

### 플랜 B: 품질점수 최적화 패키지 (대행 전환 없이 이용 가능)
| 항목 | 내용 |
|------|------|
| **대상** | 기존 대행사 유지하면서 품질점수만 올리고 싶은 광고주 |
| **방식** | 키워드별 맞춤 랜딩페이지 제작 → 품질점수 상승 → CPC 20~40% 절감 |
| **단가** | 5키워드 30만 / 10키워드 50만 / 20키워드 80만 |
| **결제** | 현금 또는 포인트 |
| **효과** | 월 광고비 300만 기준 60~120만 절약 → 1~2개월이면 제작비 회수 |
| **대행 전환 필요** | ❌ 없음 — 기존 대행사 그대로 유지 |

> 진입 장벽 제로. 대행 안 바꿔도 됨 → 먼저 관계 만들고 → 성과 확인 후 대행 전환 유도

### 플랜 C: 비교 테스트 모드
| 항목 | 내용 |
|------|------|
| **대상** | 기존 대행사와 성과 비교 후 전환 검토하고 싶은 광고주 |
| **방식** | 네이버 계정 추가 생성 (동일 사업자번호 가능) → 예산 20% 분리 → 동일 키워드 대조 테스트 |
| **기간** | 4주 테스트 → CPC/CTR/전환 수치 비교 리포트 제공 |
| **비용** | 무료 (테스트 기간 대행 수수료만) |
| **전환** | 성과 확인 → 전체 예산 이전 → 플랜 A 전환 |

> 숫자로 증명 → "전체 예산 옮기시겠어요?" 자연스러운 전환

### 3단계 고객 확보 퍼널
```
① 진입 (허들 제로) — 플랜 B
   "대행 안 바꿔도 됩니다. 랜딩페이지만 만들어드릴게요"
   → 품질점수 패키지 현금 결제
   
② 비교 테스트 — 플랜 C
   "계정 하나 더 만들어서 예산 20%만 테스트해보세요"
   → 온다 계정으로 AI 최적화 효과 증명
   
③ 전환 — 플랜 A
   "숫자 보셨죠? 전체 예산 옮기시면 무료 + 포인트 환급"
   → 대행 전환 + 온다 애드 풀 서비스
```

## 5. 핵심 기능 8가지

### 5-1. 통합 CPC 대시보드
- 네이버 검색광고 API 연동 (고객이 API 키 입력)
- 캠페인/광고그룹/키워드별 실시간 데이터
- KPI 카드: 클릭, 소진액, CPC 평균, CTR, ROAS
- 일별/주별/월별 추이 그래프
- 비교 테스트 모드: 기존 대행사 vs 온다 성과 나란히 비교

### 5-2. 부정클릭 탐지 + 환불
- 웹사이트 CPC: JS 트래킹 코드 → 방문자 핑거프린팅으로 직접 탐지
- 플레이스 CPC: API 데이터 기반 이상 패턴 탐지 (시간대별 클릭 급증, 전환율 괴리)
- 판단 기준: 동일IP 반복, 체류 3초 미만, CTR 급등, 심야 집중, 전환 0
- 부정클릭 판정 → 환불 증빙 PDF 자동 생성 → 환불 신청 대행
- **환불 수수료 무료**

### 5-3. AI 소재 자동 생성 + 자동 교체
- 저성과 소재 자동 탐지 (CTR 하위 20%)
- AI 배너/썸네일 이미지 자동 생성
- AI 광고 제목/설명문 A/B 테스트 소재 생성
- **API로 소재 자동 교체 (POST/PUT /ncc/ads)** — 사람 개입 0
- 성과 비교 → 우수 소재 자동 적용, 저성과 자동 폐기

### 5-4. CPC 단가 인하 (품질점수 최적화)
- **키워드별 맞춤 랜딩페이지 자동 생성**
  - 현재 대부분: 키워드 10개 → 랜딩 1개
  - 우리: 키워드 10개 → 맞춤 랜딩 10개
  - → 키워드-랜딩 관련성 ↑ → 품질점수 ↑ → **CPC 20~40% 절감**
- 상세페이지 세분화 + SEO 최적화
- 변경 전후 CPC 비교 리포트 제공
- **플랜 B 단독 이용 가능** (대행 전환 없이)

### 5-5. 입찰가 자동 최적화
- ROAS 기반 자동 입찰 조정
- 전환 높은 시간대 입찰 ↑, 낮은 시간대 ↓
- 저성과 키워드 자동 일시중지
- 예산 재배분 추천

### 5-6. 키워드 확장 (롱테일 자동 발굴 + 자동 등록)
- 네이버 키워드 도구 API + AI 확장 조합
- 경쟁 강도 / 예상 CPC / 예상 CTR 사전 분석
- **발굴 → API로 자동 등록 (POST /ncc/keywords) → 소액 테스트 → ROI 확인 → 자동 유지/제거**
- **맞춤 랜딩 + 맞춤 소재도 자동 생성 후 연결** (키워드+랜딩+소재 풀세트 자동화)
- 등록 시 보수적 입찰가 설정 (예상 CPC의 80%)
- 7일 후 전환 0 → 자동 삭제, ROI 양수 → 유지 + 입찰 상향

### 5-7. 소재+랜딩+키워드 풀 자동화 파이프라인
- **롱테일 100개를 키워드+랜딩+소재 풀세트로 등록하는 데 1시간**
- 수동으로 하면 일주일 걸리는 일
- 플로우:
  1. AI 롱테일 발굴 → 사전 분석
  2. 맞춤 랜딩페이지 AI 자동 생성 → Cloudflare 배포
  3. AI 소재 자동 생성 (제목/설명/이미지)
  4. API로 키워드 등록 + 소재 연결 + 랜딩 URL 설정
  5. 테스트 → 성과 측정 → 자동 정리/유지

### 5-8. ROAS 극대화
- 부정클릭 환불 → 실질 광고비 절감
- 소재 최적화 → CTR 상승
- 품질점수 최적화 → CPC 하락
- 대시보드에 "이번 달 절감액" 실시간 표시
- 주간 리포트 자동 발송

## 6. 고객 여정 (User Flow)

### 플랜 A: 풀 서비스 (대행 전환)
```
① 네이버로 로그인
   ↓
② API 키 입력 (검색광고 → 도구 → API 관리 → 복사 → 붙여넣기)
   ↓
③ 대행 계약 체결
   ↓
④ 대시보드 자동 세팅 + AI 분석 시작
   - 캠페인/키워드/소재 전부 로드
   - 부정클릭 트래킹 코드 발급
   - 저성과 소재 AI 자동 교체
   - 롱테일 키워드 자동 발굴+등록
   ↓
⑤ 고객은 대시보드에서 확인만
   - 주간 리포트 자동 발송
   - 부정클릭 알림 실시간
   - 포인트 적립 현황 확인
```

### 플랜 B: 품질점수 패키지 (대행 유지)
```
① 네이버로 로그인
   ↓
② 품질점수 패키지 신청 (5/10/20 키워드 선택)
   ↓
③ 키워드 리스트 제출
   ↓
④ AI가 키워드별 맞춤 랜딩페이지 자동 생성 → 배포
   ↓
⑤ 랜딩 URL 전달 → 기존 대행사에서 연결URL 교체
   ↓
⑥ 품질점수 상승 확인 리포트 제공 (변경 전후 CPC 비교)
```

### 플랜 C: 비교 테스트 모드
```
① 네이버로 로그인
   ↓
② API 키 입력 + 비교 테스트 신청
   ↓
③ 네이버 계정 추가 생성 안내 (동일 사업자번호 가능)
   ↓
④ 예산 20% 분리 → 동일 키워드 대조 테스트
   - 온다 계정: AI 소재 + 맞춤 랜딩 + 자동 최적화
   - 기존 계정: 기존 대행사 운영 그대로
   ↓
⑤ 4주 후 CPC/CTR/전환 비교 리포트
   ↓
⑥ 성과 확인 → 전체 예산 이전 (플랜 A 전환)
```

## 8. 자주 묻는 질문

**Q. 정말 무료인가요?**
네. 솔루션 사용료는 0원입니다. 검색광고 대행 계약만 체결해주시면 됩니다.

**Q. 기존 대행사를 바꾸고 싶지 않은데요?**
품질점수 패키지(플랜 B)는 대행 전환 없이 이용 가능합니다. 랜딩페이지만 제작해서 CPC를 낮출 수 있습니다. 또는 비교 테스트(플랜 C)로 예산 20%만 테스트해보실 수 있습니다.

**Q. 비교 테스트는 어떻게 하나요?**
같은 사업자번호로 네이버 검색광고 계정을 하나 더 만들 수 있습니다. 예산 20%를 분리해서 동일 키워드로 4주간 대조 테스트 후 성과 비교 리포트를 드립니다.

**Q. API 키 발급이 어렵진 않나요?**
검색광고 로그인 → 도구 → API 관리에서 3분이면 됩니다. 어려우시면 카카오톡으로 화면 공유하면서 도와드립니다.

**Q. 부정클릭 환불 수수료가 있나요?**
없습니다. 탐지부터 환불 신청까지 전부 무료입니다.

**Q. 환급 포인트는 현금으로 받을 수 있나요?**
포인트는 홈페이지 제작, 플레이스 마케팅, 셀프마케팅 트래픽 등 서비스에 사용 가능합니다. 현금 환급은 불가합니다.

**Q. 소재나 키워드를 자동으로 바꿔도 되나요?**
모든 자동 최적화는 대시보드에서 사전 승인/자동 모드를 선택할 수 있습니다. 자동 모드: AI가 알아서 교체. 승인 모드: AI가 제안 → 고객 확인 후 적용.

**Q. 최소 계약 기간이 있나요?**
없습니다. 언제든 해지 가능합니다.

**Q. 플레이스 광고도 관리해주나요?**
네. 플레이스 CPC 부정클릭도 API 데이터 분석으로 탐지하고, 환불 신청까지 대행합니다.

## 9. 기술 구조

| 구분 | 기술 |
|------|------|
| 프론트엔드 | HTML/CSS/JS (SPA) |
| 백엔드 | Cloudflare Workers |
| DB | Supabase |
| 인증 | 네이버 소셜 로그인 (OAuth 2.0) |
| API | 네이버 검색광고 API (고객 키) |

| 소재 자동화 | POST/PUT /ncc/ads (API 직접 교체) |
| 키워드 자동화 | POST/DELETE /ncc/keywords (자동 등록/삭제) |
| 랜딩 자동화 | AI 템플릿 생성 → Cloudflare Pages 자동 배포 |
| 환불 자동화 | 실제 브라우저 기반 자동 접수 |
| 호스팅 | Cloudflare Pages |
