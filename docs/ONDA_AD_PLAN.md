# 온다 애드 (Onda Ad) — 사업 기획서

## 1. 서비스 개요

**한줄 정의**: 네이버 검색광고 올인원 AI 관리 솔루션 (무료 SaaS)

**포지션**: 보라웨어(부정클릭) + 애드부스트(소재 최적화) + 리포트마스터(대시보드)를 하나로

**타겟**: 네이버 검색광고 운영 중인 자영업자 · 중소기업 · 병의원

## 2. 비즈니스 모델

### 수익구조
| 항목 | 내용 |
|------|------|
| **구독료** | 무료 (0원) |
| **조건** | 네이버 검색광고 대행사 권한 이전 |
| **수익원** | 대행사 요율 14% (네이버 공식 요율) |
| **크로스셀** | 홈페이지 제작 · 플레이스 마케팅 · 랜딩페이지 최적화 |

### 환급 포인트 시스템
| 월 소진액 | 환급률 | 사용처 |
|----------|--------|--------|
| 100만 미만 | 3% | 홈페이지 제작, 플레이스 마케팅 |
| 100만~500만 | 5% | 위 + 랜딩페이지 제작, 소재 디자인 |

> 포인트 명목으로 지급 (요율 환급이 아닌 마케팅 보너스 포인트)

### 홈페이지 제작 단가 (포인트 결제 가능)
| 패키지 | 가격 | 구성 | 기간 |
|--------|------|------|------|
| STANDARD | 12만 | 1페이지 + CMS + 수정무제한 | 3일 |
| DELUXE | 20만 | 메인+서브2P + CMS | 5일 |
| PREMIUM | 35만 | 메인+서브5P + CMS + 유지보수1개월 | 7일 |

> 예: 월 소진액 300만 광고주 → 월 15만 포인트 → 2개월이면 PREMIUM 무료

## 3. 핵심 기능 7가지

### 3-1. 통합 CPC 대시보드
- 네이버 검색광고 API 연동 (고객이 API 키 입력)
- 캠페인/광고그룹/키워드별 실시간 데이터
- KPI 카드: 클릭, 소진액, CPC 평균, CTR, ROAS

### 3-2. 부정클릭 탐지 + 환불 증빙 자동생성
- JS 트래킹 코드 → 방문자 핑거프린팅
- 판단 기준: 동일IP 3회+, 체류 3초 미만, CTR 300%+ 급등, 심야 집중 등
- 부정클릭 판정 → 환불 신청용 증빙 PDF 자동 생성
- 환불 신청: 로컬 PC + OpenClaw 노드로 실제 브라우저 자동 접수

### 3-3. AI 소재 생성
- 저성과 소재 자동 탐지 (CTR 하위 20%)
- GPT Image: 배너/썸네일 자동 생성
- AI 카피라이팅: 제목/설명문 A/B 테스트 소재 생성
- 네이버 검색광고 API로 소재 직접 교체

### 3-4. CPC 단가 인하 (품질점수 최적화)
- **키워드별 맞춤 랜딩페이지 자동 생성**
  - 현재: 키워드 10개 → 랜딩 1개
  - 우리: 키워드 10개 → 맞춤 랜딩 10개
  - → 키워드-랜딩 관련성 ↑ → 품질점수 ↑ → CPC 20~40% 절감
- 상세페이지 세분화 + SEO 최적화
- 효과 측정: 변경 전후 CPC 비교 대시보드

### 3-5. 데이터 분석 + 자동 수정
- 입찰가 자동 조정 (ROAS 기반)
- 저성과 키워드 일시중지 / 예산 재배분
- 시간대별 입찰 조정 (전환 높은 시간대 ↑)

### 3-6. 키워드 확장 (롱테일 자동 발굴)
- 네이버 키워드 도구 API + AI 확장
- 경쟁 강도 / 예상 CPC / 예상 CTR 사전 분석
- 발굴 → 테스트 → ROI 확인 → 자동 유지/제거

### 3-7. 환불로 ROAS 상승
- 부정클릭 환불액 → 실질 ROAS 상승
- 월간 환불 리포트 자동 생성
- 대시보드에 "이번 달 절감액" 실시간 표시

## 4. 고객 여정 (User Flow)

```
① 네이버 로그인 (네이버 소셜 로그인 필수)
   ↓
② API 키 입력 (네이버 검색광고 API — 고객이 직접 발급/입력)
   ※ API 키 자동 추출 가능하면 추가 구현
   ↓
③ 대행사 권한 이전 안내 (온라인 서명 또는 카카오톡 확인)
   ↓
④ 대시보드 자동 세팅
   - 캠페인/키워드/소재 전부 로드
   - 부정클릭 트래킹 코드 발급 → 랜딩페이지에 삽입 안내
   ↓
⑤ AI 자동 분석 시작
   - 부정클릭 탐지 → 환불 증빙
   - 저성과 소재 → AI 대체 소재 생성
   - 키워드-랜딩 불일치 → 맞춤 랜딩 자동 생성 제안
   - 입찰가 최적화 → ROAS 극대화
   ↓
⑥ 고객은 대시보드에서 확인만
   - 주간 리포트 자동 발송 (이메일/카카오톡)
   - 부정클릭 알림 실시간
   - 포인트 적립 현황 확인
```

## 5. 로그인 / 인증 구조

| 항목 | 방식 |
|------|------|
| **최초 로그인** | 네이버 소셜 로그인 (필수, 유일) |
| **API 키 입력** | 네이버 검색광고 API 키 (고객 직접 발급 → 입력) |
| **API 키 자동추출** | 가능하면 추가 (광고주센터 연동 후 자동) |
| **대행사 권한** | 별도 안내 → 광고주센터에서 권한 이전 |

### API 키 발급 안내 (고객용)
1. 네이버 검색광고 (searchad.naver.com) 로그인
2. 도구 → API 사용 관리 → API 키 발급
3. API 라이선스 키, 시크릿 키, 고객 ID 복사
4. 온다 애드 대시보드에 붙여넣기
5. 연동 완료 (1분 내)

## 7. 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | HTML/CSS/JS (SPA) 또는 Next.js |
| 백엔드 | Cloudflare Workers + D1/KV |
| DB | Supabase (고객 데이터, 분석 로그) |
| 인증 | 네이버 소셜 로그인 (OAuth 2.0) |
| API | 네이버 검색광고 API (고객 키) |
| AI | OpenAI GPT-4o (소재 카피) + GPT Image (배너) |
| 부정클릭 | JS 트래킹 코드 + 자체 분석 엔진 |
| 환불 자동화 | 로컬 PC + OpenClaw 노드 (실제 브라우저) |
| 호스팅 | Cloudflare Pages |
