# 온다 애드 (Onda Ad) — 서비스 기획서

## 1. 서비스 개요

**한줄 정의**: 네이버 검색광고 올인원 AI 관리 솔루션

**포지션**: 부정클릭 방지 + 소재 최적화 + CPC 절감 + 통합 대시보드를 하나로

**타겟**: 네이버 검색광고 운영 중인 자영업자 · 중소기업 · 병의원

## 3. 환급 포인트 시스템

광고를 많이 쓸수록 포인트가 쌓입니다.

| 월 소진액 | 환급률 | 예시 |
|----------|--------|------|
| 100만 미만 | 3% | 월 80만 소진 → 2.4만P |
| 100만~500만 | 5% | 월 300만 소진 → 15만P |
| 500만 이상 | 7% | 월 800만 소진 → 56만P |

### 포인트 사용처
- 홈페이지 제작
- 플레이스 마케팅
- 랜딩페이지 제작
- 소재 디자인
- **셀프마케팅 트래픽 구매** (플레이스 트래픽, 블로그 트래픽 등)
- 기타 전체 서비스

## 5. 핵심 기능 7가지

### 5-1. 통합 CPC 대시보드
- 네이버 검색광고 API 연동 (고객이 API 키 입력)
- 캠페인/광고그룹/키워드별 실시간 데이터
- KPI 카드: 클릭, 소진액, CPC 평균, CTR, ROAS
- 일별/주별/월별 추이 그래프

### 5-2. 부정클릭 탐지 + 환불
- 웹사이트 CPC: JS 트래킹 코드 → 방문자 핑거프린팅으로 직접 탐지
- 플레이스 CPC: API 데이터 기반 이상 패턴 탐지 (시간대별 클릭 급증, 전환율 괴리)
- 판단 기준: 동일IP 반복, 체류 3초 미만, CTR 급등, 심야 집중, 전환 0
- 부정클릭 판정 → 환불 증빙 PDF 자동 생성 → 환불 신청 대행
- **환불 수수료 무료**

### 5-3. AI 소재 생성
- 저성과 소재 자동 탐지 (CTR 하위 20%)
- AI 배너/썸네일 이미지 자동 생성
- AI 광고 제목/설명문 A/B 테스트 소재 생성
- 성과 비교 → 우수 소재 자동 적용

### 5-4. CPC 단가 인하 (품질점수 최적화)
- **키워드별 맞춤 랜딩페이지 자동 생성**
  - 현재 대부분: 키워드 10개 → 랜딩 1개
  - 우리: 키워드 10개 → 맞춤 랜딩 10개
  - → 키워드-랜딩 관련성 ↑ → 품질점수 ↑ → **CPC 20~40% 절감**
- 상세페이지 세분화 + SEO 최적화
- 변경 전후 CPC 비교 리포트 제공

### 5-5. 입찰가 자동 최적화
- ROAS 기반 자동 입찰 조정
- 전환 높은 시간대 입찰 ↑, 낮은 시간대 ↓
- 저성과 키워드 자동 일시중지
- 예산 재배분 추천

### 5-6. 키워드 확장 (롱테일 자동 발굴 + 자동 등록)
- 네이버 키워드 도구 API + AI 확장 조합
- 경쟁 강도 / 예상 CPC / 예상 CTR 사전 분석
- **발굴 → API로 자동 등록 (POST /ncc/keywords) → 소액 테스트 → ROI 확인 → 자동 유지/제거**
- 등록 시 보수적 입찰가 설정 (예상 CPC의 80%)
- 7일 후 전환 0 → 자동 삭제, ROI 양수 → 유지 + 입찰 상향

### 5-7. ROAS 극대화
- 부정클릭 환불 → 실질 광고비 절감
- 소재 최적화 → CTR 상승
- 품질점수 최적화 → CPC 하락
- 대시보드에 "이번 달 절감액" 실시간 표시
- 주간 리포트 자동 발송

## 6. 고객 여정 (User Flow)

```
① 네이버로 로그인 (네이버 소셜 로그인)
   ↓
② API 키 입력
   (검색광고 → 도구 → API 관리 → 복사 → 붙여넣기)
   ※ 자동추출 기능 추가 예정
   ↓
③ 대행 계약 안내 (온라인 서명 또는 카카오톡 확인)
   ↓
④ 대시보드 자동 세팅
   - 캠페인/키워드/소재 전부 로드
   - 부정클릭 트래킹 코드 발급
   ↓
⑤ AI 자동 분석 시작
   - 부정클릭 탐지 → 환불 증빙
   - 저성과 소재 → AI 대체 소재 생성
   - 키워드-랜딩 불일치 → 맞춤 랜딩 제안
   - 입찰가 최적화
   ↓
⑥ 고객은 대시보드에서 확인만
   - 주간 리포트 자동 발송
   - 부정클릭 알림 실시간
   - 포인트 적립 현황 확인
```

## 8. 자주 묻는 질문

**Q. 정말 무료인가요?**
네. 솔루션 사용료는 0원입니다. 검색광고 대행 계약만 체결해주시면 모든 기능을 무료로 사용하실 수 있습니다.

**Q. 기존 대행사가 있는데요?**
대행사 변경은 광고주센터에서 간단히 가능합니다. 기존 캠페인/키워드/소재 전부 그대로 유지됩니다.

**Q. API 키 발급이 어렵진 않나요?**
검색광고 로그인 → 도구 → API 관리에서 3분이면 됩니다. 어려우시면 카카오톡으로 화면 공유하면서 도와드립니다.

**Q. 부정클릭 환불 수수료가 있나요?**
없습니다. 탐지부터 환불 신청까지 전부 무료입니다.

**Q. 환급 포인트는 현금으로 받을 수 있나요?**
포인트는 홈페이지 제작, 플레이스 마케팅 등 서비스에 사용 가능합니다. 현금 환급은 불가합니다.

**Q. 최소 계약 기간이 있나요?**
없습니다. 언제든 해지 가능합니다.
