## 이런 분께 추천합니다

- 네이버 검색광고 운영 중인데 **부정클릭 때문에 광고비가 새는** 분
- 소재 만들 시간이 없어서 **같은 배너를 1년째 쓰고 있는** 분
- CPC가 올라만 가는데 **뭘 어떻게 해야 할지 모르는** 분
- 부정클릭 솔루션/소재 최적화/리포팅 **따로따로 쓰는 게 번거로운** 분
- **광고비 절감**은 하고 싶은데 별도 솔루션 비용이 부담인 분

## 환급 포인트

광고를 많이 쓸수록 포인트가 쌓입니다.

| 월 광고 소진액 | 환급률 | 예시 |
|---------------|--------|------|
| 100만 미만 | 3% | 월 80만 소진 → 2.4만P |
| 100만~500만 | 5% | 월 300만 소진 → 15만P |
| 500만 이상 | 7% | 월 800만 소진 → 56만P |

### 포인트 사용처
- 홈페이지 제작 (12만~35만)
- 플레이스 마케팅
- 랜딩페이지 제작
- 소재 디자인
- 기타 전체 서비스

> 예: 월 300만 소진 → 2개월이면 30만P → PREMIUM 홈페이지(35만) 거의 무료

## 포함 기능

### 1. 통합 대시보드
네이버 검색광고 모든 데이터를 한 화면에서 확인합니다.
- 캠페인/광고그룹/키워드별 클릭, 소진액, CPC, CTR, 전환수
- 일별/주별/월별 추이 그래프
- 핵심 KPI 한눈에 확인

### 2. 부정클릭 방지 + 환불
경쟁사의 악의적 클릭, 봇 클릭을 잡아냅니다.
- **웹사이트 CPC**: 트래킹 코드로 방문자 직접 분석 (IP, 디바이스, 체류시간)
- **플레이스 CPC**: API 데이터 기반 이상 패턴 탐지 (클릭 급증 + 전환 0 등)
- 동일 IP 반복, 체류 3초 미만, CTR 급등, 심야 비정상 클릭 탐지
- 환불 증빙 자동 생성 → 환불 신청 대행
- **환불 수수료 0원**

### 3. AI 소재 생성
클릭률 낮은 소재를 AI가 자동으로 만들어 교체합니다.
- AI 배너/썸네일 이미지 생성
- AI 광고 제목/설명문 작성
- A/B 테스트용 소재 자동 세팅
- 성과 비교 후 우수 소재 자동 적용

### 4. CPC 단가 인하
같은 순위를 더 싸게 가져갑니다.
- 키워드별 맞춤 랜딩페이지 자동 생성
- 키워드-랜딩 관련성 향상 → 품질점수 상승 → **CPC 20~40% 절감**
- 상세페이지 세분화 + SEO 최적화
- 변경 전후 CPC 비교 리포트 제공

### 5. 입찰가 자동 최적화
- ROAS 기반 자동 입찰 조정
- 전환 높은 시간대 ↑, 낮은 시간대 ↓
- 저성과 키워드 자동 일시중지
- 예산 재배분 추천

### 6. 키워드 확장
새로운 롱테일 키워드를 AI가 발굴합니다.
- 경쟁 강도 / 예상 CPC / 예상 CTR 사전 분석
- 발굴 → 소액 테스트 → ROI 확인 → 자동 유지/제거

### 7. ROAS 극대화
- 부정클릭 환불로 실질 광고비 절감
- 소재 최적화로 CTR 상승
- 품질점수 최적화로 CPC 하락
- "이번 달 절감액" 실시간 표시
- 주간 리포트 자동 발송

## 시작 방법 (3분)

1. **네이버로 로그인**
2. **API 키 입력** (검색광고 → 도구 → API 관리 → 복사 → 붙여넣기)
3. **대행 계약 체결** (안내에 따라 간단 서명)
4. **끝.** 대시보드에서 바로 확인 가능

## 자주 묻는 질문

**Q. 정말 무료인가요?**
네. 솔루션 사용료는 0원입니다. 검색광고 대행 계약만 체결해주시면 됩니다.

**Q. 기존 대행사가 있는데요?**
대행사 변경은 광고주센터에서 간단히 가능합니다. 기존 캠페인/키워드/소재 전부 그대로 유지됩니다.

**Q. API 키 발급이 어렵진 않나요?**
검색광고 로그인 → 도구 → API 관리에서 3분이면 됩니다. 어려우시면 카카오톡으로 화면 공유하면서 도와드립니다.

**Q. 부정클릭 환불 수수료가 있나요?**
없습니다. 탐지부터 환불 신청까지 전부 무료입니다.

**Q. 환급 포인트는 현금으로 받을 수 있나요?**
포인트는 홈페이지 제작, 플레이스 마케팅 등 서비스에 사용 가능합니다. 현금 환급은 불가합니다.

**Q. 최소 계약 기간이 있나요?**
없습니다. 언제든 해지 가능합니다.

**Q. 플레이스 광고도 관리해주나요?**
네. 플레이스 CPC 부정클릭도 API 데이터 분석으로 탐지하고, 이상 패턴 발견 시 환불 신청까지 대행합니다.
