# 온다 애드 — 상품 설명서 (크몽 서비스 페이지용)

---

## 상품명
**네이버 검색광고 올인원 AI 관리** — 부정클릭 방지 · 소재 최적화 · CPC 절감

---

## 한줄 요약
보라웨어 + 애드부스트 + 리포트마스터를 하나로. **무료 제공.**

---

## 이런 분께 추천합니다

- 네이버 검색광고 운영 중인데 **부정클릭 때문에 광고비가 새는** 분
- 소재 만들 시간이 없어서 **같은 배너를 1년째 쓰고 있는** 분
- CPC가 올라만 가는데 **뭘 어떻게 해야 할지 모르는** 분
- 보라웨어/애드부스트 **따로따로 쓰는 게 번거로운** 분
- **광고비 절감**은 하고 싶은데 대행사 비용이 부담인 분

| 항목 | 내용 |

## 환급 포인트

광고를 많이 쓸수록 포인트가 쌓입니다.

| 월 광고 소진액 | 환급률 | 예시 |
|---------------|--------|------|
| 100만 미만 | 3% | 월 80만 소진 → 2.4만P |
| 100만~500만 | 5% | 월 300만 소진 → 15만P |
| 500만 이상 | 7% | 월 800만 소진 → 56만P |

### 포인트 사용처
- 홈페이지 제작 (12만~35만)
- 플레이스 마케팅
- 랜딩페이지 제작
- 소재 디자인
- 기타 온다마케팅 서비스 전체

> 예: 월 300만 소진 광고주 → 2개월이면 30만P → PREMIUM 홈페이지(35만) 거의 무료

| 패키지 | 가격 | 구성 | 기간 |

호스팅: 무료 (우리 서버 제공, 도메인만 연 1~2만 별도)

## 포함 기능 상세

### 1. 통합 대시보드
네이버 검색광고 모든 데이터를 한 화면에.
- 캠페인/광고그룹/키워드별 클릭, 소진액, CPC, CTR, 전환수
- 일별/주별/월별 추이 그래프
- KPI 카드: 핵심 지표 한눈에

### 2. 부정클릭 방지
경쟁사의 악의적 클릭, 봇 클릭을 잡아냅니다.
- 동일 IP 반복 클릭 탐지
- 체류 3초 미만 즉시 이탈 탐지
- VPN/프록시 IP 블랙리스트
- CTR 이상 급등 감지
- 심야 시간대 비정상 클릭 탐지
- **환불 증빙 PDF 자동 생성** → 네이버에 환불 신청

### 3. AI 소재 생성
클릭률 낮은 소재를 AI가 자동으로 만들어 교체합니다.
- AI 배너/썸네일 이미지 생성
- AI 광고 제목/설명문 작성
- A/B 테스트용 소재 자동 세팅
- 성과 비교 → 우수 소재 자동 적용

### 4. CPC 단가 인하
같은 순위를 더 싸게 가져갑니다.
- **키워드별 맞춤 랜딩페이지** 자동 생성
  - "강남 정형외과" → 강남 특화 랜딩
  - "무릎 통증 병원" → 무릎 전문 랜딩
- 키워드-랜딩 관련성 ↑ → 품질점수 ↑ → **CPC 20~40% 절감**
- 변경 전후 CPC 비교 리포트 제공

### 5. 입찰가 자동 최적화
- ROAS 기반 자동 입찰 조정
- 전환 높은 시간대 입찰 ↑, 낮은 시간대 ↓
- 저성과 키워드 자동 일시중지
- 예산 재배분 추천

### 6. 키워드 확장
새로운 롱테일 키워드를 AI가 발굴합니다.
- 경쟁 강도 / 예상 CPC / 예상 CTR 사전 분석
- 발굴 → 소액 테스트 → ROI 확인 → 자동 유지/제거
- "이 키워드 추가하면 월 전환 +12건 예상" 식의 구체적 제안

### 7. ROAS 극대화
- 부정클릭 환불 → 실질 광고비 절감
- 소재 최적화 → CTR 상승
- 품질점수 최적화 → CPC 하락
- 종합 효과: **같은 예산으로 전환 2배** 목표

## 시작 방법 (3분)

1. **네이버로 로그인**
2. **API 키 입력** (검색광고 → 도구 → API 관리 → 복사 → 붙여넣기)
3. **대행사 권한 이전** (안내에 따라 클릭 몇 번)
4. **끝.** 대시보드에서 바로 확인 가능

## 자주 묻는 질문

**Q. 정말 무료인가요?**
A. 네. 솔루션 사용료는 0원입니다. 네이버 검색광고 대행사 권한만 이전해주시면 됩니다. 대행사 요율 14%는 네이버 공식 요율로, 어떤 대행사를 쓰든 동일하게 적용됩니다.

**Q. 기존 대행사가 있는데요?**
A. 대행사 변경은 광고주센터에서 간단히 가능합니다. 기존 캠페인/키워드/소재 전부 그대로 유지됩니다. 변경 방법도 안내해드립니다.

**Q. API 키 발급이 어렵진 않나요?**
A. 검색광고 로그인 → 도구 → API 관리에서 3분이면 됩니다. 스크린샷 가이드 제공하고, 어려우시면 카카오톡으로 화면 공유하면서 도와드립니다.

**Q. 부정클릭 환불은 어떻게 되나요?**
A. 부정클릭이 탐지되면 증빙 리포트를 자동 생성하고, 네이버에 환불 신청까지 대행합니다. 환불 성공 시 별도 수수료 없습니다.

**Q. 환급 포인트는 현금으로 받을 수 있나요?**
A. 포인트는 온다마케팅 서비스(홈페이지 제작, 플레이스 마케팅 등)에 사용 가능합니다. 현금 환급은 불가합니다.

**Q. 최소 계약 기간이 있나요?**
A. 없습니다. 언제든 해지 가능합니다.
