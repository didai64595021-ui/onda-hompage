# 네이버 검색 API 발급 가이드

## 1단계: 네이버 개발자센터 접속

아래 링크로 접속해주세요.

🔗 https://developers.naver.com

> 네이버 계정으로 로그인해주세요.

---

## 2단계: 애플리케이션 등록

1. 상단 메뉴에서 **[Application]** → **[애플리케이션 등록]** 클릭
2. 아래 내용을 입력해주세요:

| 항목 | 입력값 |
|------|--------|
| 애플리케이션 이름 | 원하시는 이름 (예: 우리가게검색) |
| 사용 API | ✅ **검색** 선택 |
| 환경 | **WEB 설정** 선택 → URL에 `http://localhost` 입력 |

3. **[등록하기]** 버튼 클릭

---

## 3단계: API 키 확인

등록이 완료되면 **[내 애플리케이션]** 페이지에서 아래 2개 값을 확인하실 수 있습니다.

- **Client ID** (공개키)
- **Client Secret** (비밀키)

> ⚠️ Client Secret은 외부에 노출되지 않도록 주의해주세요.

---
