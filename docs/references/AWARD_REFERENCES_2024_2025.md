## 🎨 2025 디자인 트렌드 & CSS 기법

### 핵심 트렌드
1. **Scroll-Driven Animations** (CSS Only)
   - `animation-timeline: scroll()` — 스크롤 위치에 연동
   - `animation-timeline: view()` — 뷰포트 진입 시 트리거
   - JS 없이 부드러운 reveal/fade/slide 가능

2. **CSS Parallax** (No JS)
   - `perspective` + `translateZ()` + `scale()` 조합
   - 다층 레이어로 깊이감 연출
   - `background-attachment: fixed` (심플 패럴랙스)

3. **Micro-interactions**
   - 버튼 hover: 색상/크기/그림자 변화 (300ms 이내)
   - 폼 필드: focus 시 하이라이트
   - 메뉴 아이콘: 햄버거 → X 모핑
   - 로딩: 스켈레톤 스크린 > 스피너

4. **Scrollytelling**
   - 스크롤로 스토리 전개
   - Sticky 섹션 + 콘텐츠 전환
   - 비선형 스크롤 (가로/대각선)

5. **Typography as Design**
   - 대형 타이포 히어로
   - 가변 폰트 (Variable Fonts)
   - 텍스트 마스크/클리핑

6. **Container Queries**
   - 부모 크기 기반 반응형 (뷰포트 X)
   - 모듈형 컴포넌트 완벽 대응

7. **`:has()` Selector**
   - 자식 기반 부모 스타일링
   - 조건부 레이아웃 변경

### CSS 코드 패턴 (복사 가능)

```css
/* 스크롤 기반 Reveal */
.reveal-on-scroll {
  animation: fadeInUp 1s ease both;
  animation-timeline: view();
  animation-range: entry 10% cover 40%;
}
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(40px); }
  to { opacity: 1; transform: translateY(0); }
}

/* CSS Only 패럴랙스 */
.parallax-container {
  perspective: 1px;
  height: 100vh;
  overflow-x: hidden;
  overflow-y: auto;
}
.parallax-bg {
  transform: translateZ(-1px) scale(2);
}
.parallax-fg {
  transform: translateZ(0);
}

/* 스크롤 프로그레스 바 */
.progress-bar {
  position: fixed;
  top: 0;
  left: 0;
  height: 3px;
  background: var(--accent);
  transform-origin: left;
  animation: grow linear;
  animation-timeline: scroll();
}
@keyframes grow {
  from { transform: scaleX(0); }
  to { transform: scaleX(1); }
}

/* 마이크로 인터랙션 버튼 */
.btn-micro {
  transition: all 0.25s cubic-bezier(0.4, 0, 0.2, 1);
}
.btn-micro:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 24px rgba(0,0,0,0.15);
}
.btn-micro:active {
  transform: translateY(0);
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

/* 텍스트 마스크 (배경 이미지) */
.text-mask {
  background: linear-gradient(135deg, #667eea, #764ba2);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

/* 카드 기울기 hover */
.card-tilt:hover {
  transform: perspective(1000px) rotateY(5deg) rotateX(-3deg);
  transition: transform 0.4s ease;
}
```

## 📏 퍼포먼스 규칙 (필수 준수)

1. **애니메이션은 `transform` + `opacity`만** — layout/paint 트리거 금지
2. **`will-change` 남용 금지** — 필요한 요소에만 적용
3. **`prefers-reduced-motion` 대응** — 접근성 필수
4. **이미지: WebP/AVIF + lazy loading** — LCP 최적화
5. **폰트: font-display: swap** — CLS 방지
6. **CSS 파일 크기: 50KB 이하** (Gzip 기준)
