### 피부과 디자인 패턴 분석

#### 히어로 유형 (TOP 5)
1. **스킨 클로즈업 + 미니멀 텍스트**: 깨끗한 피부 사진 + 시술명/CTA (가장 효과적)
2. **비포/애프터 슬라이더 히어로**: 즉시 결과 시각화 → 전환율 최고
3. **의료진 프로필 히어로**: 의사 사진 + 자격 + "상담 예약" → 신뢰 즉시 구축
4. **비디오 히어로**: 시술 과정/클리닉 투어 짧은 영상 → 몰입감
5. **AI 스킨 분석 인터랙티브**: 사진 업로드 → 즉시 피부 분석 → CTA

#### 컬러 팔레트 (TOP 5)
1. **화이트 + 로즈골드**: 여성 타겟, 프리미엄 에스테틱 (가장 인기)
2. **크림/베이지 + 세이지 그린**: 자연주의, 클린뷰티
3. **순백 + 민트/틸**: 의료 청결감 + 현대적
4. **다크 네이비 + 골드**: 남성 포함 유니섹스 럭셔리
5. **소프트 라벤더 + 화이트**: 안티에이징, 릴렉싱

#### 필수 기능 (피부과 전용)
- 비포/애프터 갤러리 (드래그 슬라이더)
- 시술별 상세 페이지 (과정/회복기간/비용대)
- 온라인 상담 예약 (날짜+시간+시술 선택)
- 의료진 프로필 (학력/경력/전문분야)
- 실시간 채팅/카카오 상담
- 자주 묻는 질문 (시술별)
- 피부 타입 퀴즈/자가 진단

#### 인터랙션 (어워드급 필수)
1. 비포/애프터 드래그 슬라이더 (CSS + JS)
2. 시술 타임라인 (시술과정 → 회복 → 결과)
3. 피부 레이어 인포그래픽 (SVG 애니메이션)
4. 360° 시설 가상투어
5. 스크롤 기반 시술 과정 설명 (스텝 바이 스텝)
