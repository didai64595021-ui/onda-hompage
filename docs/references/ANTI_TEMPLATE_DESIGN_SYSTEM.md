## 핵심 원칙: 왜 템플릿처럼 보이는가?

### ❌ 템플릿의 공통 특징 (절대 금지)
1. **순서가 똑같다**: 히어로 → 서비스카드 3열 → 소개 → 카운터 → 후기 → CTA
2. **카드가 균일하다**: 3열 같은 크기, 같은 높이, 같은 구조
3. **여백이 균일하다**: 모든 섹션 padding 80px, gap 24px
4. **인터랙션이 없다**: 스크롤해도 아무 변화 없음
5. **컬러가 단조롭다**: 전체 동일 배경색 + 동일 액센트
6. **타이포가 평범하다**: h2 + p + button 반복

### ✅ Award급 사이트의 차이점
1. **리듬감**: 큰 섹션 → 작은 섹션 → 와이드 → 좁은 → 시각적 리듬
2. **비대칭**: 좌60%/우40%, 이미지가 그리드를 벗어남
3. **전환**: 섹션마다 다른 배경색/질감/레이아웃
4. **인터랙션**: 스크롤 시 요소가 살아남 (reveal, parallax, morph)
5. **여백 변화**: 섹션마다 다른 여백, 의도적 밀도 차이
6. **타이포 대비**: 거대한 제목 vs 작은 본문, 세리프 vs 산세리프 혼합

## 🏗️ 레이아웃 패턴 라이브러리 (20가지)

### A. 히어로 패턴 (7가지)

#### A1. 풀스크린 미디어 + 오버레이 텍스트
```
┌─────────────────────────────┐
│    (배경 이미지/영상 100%)   │
│                             │
│     큰 제목                  │
│     부제목                   │
│     [CTA 버튼]              │
│                             │
└─────────────────────────────┘
```
- 용도: 감성적 브랜드 (호텔, 레스토랑, 부동산)
- 핵심: 오버레이 그라데이션 + 텍스트 가독성

#### A2. 스플릿 히어로 (텍스트/이미지 분리)
```
┌──────────────┬──────────────┐
│  큰 제목      │              │
│  부제목       │   이미지     │
│  [CTA]       │              │
│              │              │
└──────────────┴──────────────┘
```
- 용도: 기업, 서비스, 병원
- 변형: 좌60/우40, 이미지가 화면 밖까지 확장

#### A3. 센터 타이포 히어로 (이미지 없음)
```
┌─────────────────────────────┐
│                             │
│        거대한 제목           │
│     (100px+ 타이포)          │
│        작은 부제목           │
│                             │
└─────────────────────────────┘
```
- 용도: 크리에이티브 에이전시, 포트폴리오
- 핵심: 타이포 자체가 비주얼, 배경은 미니멀/그라데이션

#### A4. 벤토 그리드 히어로
```
┌────────┬────────┬──────────┐
│ 제목   │ 이미지1 │          │
│ 부제목  │        │ 이미지2  │
├────────┤        │          │
│ [CTA]  ├────────┤          │
│ 통계   │ 이미지3 │          │
└────────┴────────┴──────────┘
```
- 용도: SaaS, 테크, 에이전시
- 핵심: 정보 밀도 높음, 다양한 콘텐츠 동시 전달

#### A5. 스크롤 트리거 히어로 (텍스트 빌드업)
```
스크롤 시:
1. 배경 등장
2. 제목 한 글자씩 타이핑
3. 이미지 fade-in
4. CTA 슬라이드업
```
- 용도: 스토리텔링 강한 브랜드
- 핵심: 시퀀셜 reveal, 극적 첫인상

#### A6. 마키 배너 히어로 (무한 스크롤 텍스트)
```
┌─────────────────────────────┐
│ ←← 브랜드명 · 슬로건 · 브랜드명 ←← │
│                             │
│   메인 이미지/일러스트         │
│                             │
│ →→ 키워드 · 키워드 · 키워드 →→  │
└─────────────────────────────┘
```
- 용도: 패션, 뷰티, 크리에이티브
- 핵심: CSS animation infinite, 역동적 느낌

#### A7. 비디오 루프 + 오버레이 UI
```
┌─────────────────────────────┐
│ [NAV]                       │
│                             │
│   짧은 MP4 루프 (muted)      │
│   위에 반투명 UI 카드 배치    │
│                             │
└─────────────────────────────┘
```
- 용도: 시공, 인테리어, 리조트

1. **주색(Primary)**: 브랜드 대표색 1개
2. **배경(Surface)**: 밝음/어둠 2종
3. **액센트(Accent)**: 주색 보색 or 유사색 1개  
4. **텍스트(Text)**: 고대비 2종 (제목/본문)

## 🔄 조합 금지 규칙

1. **같은 레시피 2번 사용 금지** — 사용 후 SITE_GENERATOR_RULES.md에 기록
2. **연속 3개+ 동일 패턴 금지** — B8→B8→B8 (지그재그 3연속) = 템플릿
3. **카드 3열 균일 → 반드시 비대칭으로** — 3열이면 크기/높이 다르게
4. **모든 섹션 같은 배경색 금지** — 최소 3개 배경색 전환
5. **인터랙션 0개 금지** — 최소 fade-in + 1개 특수효과
6. **레시피는 시작점일 뿐** — 업종/요청에 맞게 변형 필수
