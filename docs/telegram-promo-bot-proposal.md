# 텔레그램 홍보 봇 / 매크로 — 기획서 및 기능제안서

## 1. 개요

텔레그램 그룹/채널에 자동으로 홍보 메시지를 발송하는 봇 시스템.
타겟 그룹에 가입 → 메시지 발송 → 반응 모니터링까지 자동화.

---

## 2. 핵심 기능

### 2-1. 자동 메시지 발송
| 기능 | 설명 |
|---|---|
| 다중 그룹 발송 | 설정한 그룹 리스트에 순차/랜덤 발송 |
| 스케줄링 | 시간대별 자동 발송 (예: 09시, 13시, 18시) |
| 간격 조절 | 그룹 간 발송 간격 랜덤화 (30초~5분) — 차단 방지 |
| 메시지 템플릿 | 여러 템플릿 로테이션 (동일 메시지 반복 방지) |
| 미디어 지원 | 텍스트 + 이미지/영상/파일 첨부 |
| 버튼/링크 | 인라인 버튼, URL 버튼 포함 가능 |

### 2-2. 그룹 관리
| 기능 | 설명 |
|---|---|
| 그룹 자동 가입 | 초대 링크 리스트 → 자동 가입 |
| 그룹 스크래핑 | 키워드 기반 그룹 검색 + 리스트 수집 |
| 그룹 분류 | 업종/지역별 태그 관리 |
| 차단 감지 | 밴/킥 감지 → 자동 제외 |
| 가입 쿨다운 | 하루 가입 수 제한 (텔레그램 제재 방지) |

### 2-3. 계정 관리 (핵심)
| 기능 | 설명 |
|---|---|
| 멀티 계정 | 여러 텔레그램 계정 로테이션 |
| 계정 웜업 | 신규 계정 자연스러운 활동 (가입/대화/시간차) |
| 계정 상태 모니터 | 제재/차단/번호정지 실시간 감지 |
| 자동 전환 | 계정 제재 시 다른 계정으로 자동 전환 |
| 세션 관리 | .session 파일 기반 계정 저장/복원 |

### 2-4. 안티밴 시스템
| 기능 | 설명 |
|---|---|
| 발송 속도 제어 | 시간당 최대 발송 수 제한 |
| 메시지 변형 | 동의어 치환, 이모지 삽입, 줄바꿈 변형 |
| 지문 분산 | 계정별 다른 디바이스 정보 |
| 스팸 신고 감지 | 신고 임계치 도달 시 자동 중단 |
| 휴식 모드 | 일정 발송 후 강제 휴식 (1~6시간) |

### 2-5. 모니터링 & 리포트
| 기능 | 설명 |
|---|---|
| 발송 통계 | 성공/실패/차단 건수 실시간 |
| 반응 추적 | 조회수, 클릭수 (URL 트래커) |
| 일일 리포트 | 텔레그램 DM으로 자동 보고 |
| 대시보드 | 웹 대시보드 (발송현황, 계정상태, 그룹 맵) |

---

## 3. 기술 스택

```
언어: Python 3.11+ (Telethon / Pyrogram)
DB: SQLite (경량) 또는 PostgreSQL (대규모)
스케줄러: APScheduler
UI: 텔레그램 봇 커맨드 + 웹 대시보드 (Next.js)
배포: VPS (한국 IP 권장)
계정 인증: 전화번호 기반 OTP
```

## 4. 수익 모델 (판매 시)

### 패키지 구성안
| 플랜 | 기능 | 가격(월) |
|---|---|---|
| 베이직 | 1계정, 50그룹, 일 100건 발송 | 19만원 |
| 프로 | 3계정, 200그룹, 일 500건, 안티밴 | 39만원 |
| 엔터프라이즈 | 10계정+, 무제한그룹, 커스텀 | 79만원~ |

### 추가 수익
- 셋업비: 30~50만원 (초기 설정 + 계정 웜업)
- 그룹 리스트 판매: 업종별 5~10만원
- 커스텀 개발: 건당 협의

## 7. 차별화 포인트

1. **AI 메시지 생성** — Claude로 그룹 성격에 맞는 홍보 메시지 자동 생성
2. **스마트 타겟팅** — 그룹 멤버 활동 분석 → 반응 좋은 시간대 자동 선택
3. **원클릭 셋업** — 전화번호만 넣으면 가입~발송~모니터링 올인원
4. **한국어 특화** — 한국 텔레그램 그룹 DB + 한국어 메시지 최적화
