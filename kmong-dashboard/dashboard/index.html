데이터 정합성 공지 — crawl-cpc.js 날짜 필터 버그 수정 (2026-04-07)
크롤러가 "어제" 필터를 button 셀렉터로 찾으려 했으나 실제 DOM은 a.rounded-full 였습니다. 결과적으로 클릭이 실패하여 페이지 default인 "지난 7일" 누적값이 매일 "어제 데이터"로 저장되어 광고비가 약 5~9배 부풀려진 채로 14일치(약 170행) 누적되었습니다.
수정 내역 (커밋 123802f)
  • 진입 안내 모달 자동 닫기
  • a.rounded-full:has-text("어제") 명시적 클릭 + bg-gray-900 활성 클래스 검증 (실패 시 throw)
  • PRODUCT_MAP 키워드 충돌로 인한 product_id 중복 행 dedup (비용 큰 행 우선)
※ 과거 inflated 14일치 데이터는 분석 비교 베이스 보존을 위해 유지. 다음 cron 사이클부터 정상 수치로 적재.
