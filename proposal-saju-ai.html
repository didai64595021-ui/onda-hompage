Project Understanding

프로젝트 이해도

클라이언트의 핵심 니즈를 정확히 파악하여, 최적의 솔루션을 설계합니다.

폐쇄형 내부 시스템

외부 노출 없는 내부 업무 전용 웹 시스템. 관리자 인증 기반으로 데이터 보안을 최우선으로 설계합니다.

엑셀 → 시스템 이식

기존 엑셀 기반 수작업 프로세스를 체계적인 웹 시스템으로 전환하여 업무 효율성을 극대화합니다.

LLM 초안 생성

GPT-4o / Gemini Pro를 활용해 사주 분석 데이터를 기반으로 리포트 초안을 자동 생성합니다.

웹 에디터 + PDF 퍼블리싱

AI 초안을 전문가가 웹 에디터로 검수/수정 후, 고해상도 PDF로 최종 퍼블리싱합니다.

핵심 목표: 완전 자동화 파이프라인 구축

고객 데이터 입력부터 PDF 출력까지, 최소한의 수동 개입으로 동작하는 End-to-End 자동화 시스템을 구축합니다.

데이터 입력 자동화 천문 API 연동 AI 리포트 생성 WYSIWYG 에디터 PDF 자동 퍼블리싱 대시보드 관리
Tech Stack

기술 제안

검증된 모던 기술 스택으로 안정성과 확장성을 동시에 확보합니다.

Frontend
Next.js 14 + TypeScript

App Router 기반 SSR/SSG로 빠른 로딩. TypeScript로 타입 안전성 확보. TailwindCSS로 일관된 디자인 시스템 구축.

Next.js 14 TypeScript TailwindCSS
Backend
API Routes + Prisma

Next.js API Routes로 프론트-백엔드 통합. Prisma ORM으로 타입 안전한 DB 쿼리. PostgreSQL로 안정적 데이터 관리.

API Routes Prisma PostgreSQL
AI Engine
GPT-4o / Gemini Pro

명리학 도메인 특화 프롬프트 엔지니어링. 멀티 LLM 지원으로 비용 최적화. 일관된 고품질 리포트 초안 생성.

OpenAI Gemini Prompt Eng.
PDF Engine
Puppeteer + 템플릿 엔진

커스텀 HTML/CSS 템플릿 기반 고해상도 PDF 생성. 100페이지 이상 안정 출력. 동양 철학 스타일 디자인 적용.

Puppeteer Handlebars PDF/A
Infrastructure
Vercel + Supabase

Vercel로 무중단 배포 및 자동 스케일링. Supabase로 PostgreSQL + Auth + Storage 통합 관리. 서버리스 아키텍처.

Vercel Supabase Edge Fn

한국천문연구원 API 연동으로 정확한 음양력 변환 및 절기 데이터 산출. 사주 팔자 계산의 천문학적 정확성 보장.

KASI API 음양력 절기 계산

Core Features

핵심 기능 상세

8가지 핵심 모듈로 구성된 완전한 리포트 자동화 시스템입니다.

관리자 인증 시스템

Supabase Auth 기반 안전한 로그인. 역할 기반 접근 제어(RBAC)로 보안 강화.

Supabase Auth + JWT

고객 데이터 관리

생년월일, 출생시간, 성별 등 고객 정보를 체계적으로 관리하고 검색합니다.

Prisma + PostgreSQL

천문 API 연동

KASI API로 정확한 음양력 변환, 절기 계산, 천문 데이터를 실시간 연동합니다.

REST API + 캐싱

데이터 정규화 엔진

천간지지, 오행, 십성, 12운성 등 명리학 데이터를 알고리즘으로 정확히 산출합니다.

커스텀 알고리즘

AI 리포트 생성

명리학 특화 프롬프트로 GPT-4o/Gemini가 섹션별 분석 초안을 자동 생성합니다.

Multi-LLM + Streaming

WYSIWYG 웹 에디터

TipTap 기반 리치 텍스트 에디터로 AI 초안을 직관적으로 검수하고 수정합니다.

TipTap + 자동저장

PDF 퍼블리싱

커스텀 템플릿으로 100+페이지 고해상도 PDF를 안정적으로 자동 생성합니다.

Puppeteer + 템플릿

운영 대시보드

리포트 발행 현황, 고객 통계, AI 사용량 등 핵심 지표를 실시간 모니터링합니다.

Recharts + 실시간
MVP Strategy

MVP 제안 및 예산 최적화

핵심 기능 우선 개발로 빠른 가치 검증, 단계적 고도화로 예산을 최적화합니다.

Phase 1

핵심 파이프라인 구축

데이터 입력부터 PDF 출력까지, 핵심 자동화 플로우를 우선 완성합니다.

  • 관리자 로그인 및 인증 시스템
  • 고객 데이터 CRUD + 엑셀 가져오기
  • KASI API 연동 + 사주 데이터 정규화
  • AI 리포트 초안 생성 (GPT-4o)
  • 기본 웹 에디터 (검수/수정)
  • PDF 출력 (기본 템플릿 1종)
핵심 워크플로우 완성으로 즉시 업무에 투입 가능
Phase 2

고도화 및 확장

Phase 1 피드백을 반영하여 기능을 고도화하고 운영 편의성을 확보합니다.

  • 멀티 LLM 지원 (Gemini Pro 추가)
  • PDF 템플릿 다양화 (3종 이상)
  • 운영 대시보드 (통계/모니터링)
  • 리포트 히스토리 및 버전 관리
  • 배치 처리 (다건 동시 생성)
  • 에디터 고도화 (이미지/차트 삽입)
실사용 피드백 기반 고도화로 불필요한 개발 비용 절감
Timeline

개발 일정 제안

약 8~10주 일정으로, 핵심 파이프라인부터 고도화까지 단계적으로 진행합니다.

WEEK 1-2

프로젝트 셋업 & 설계

요구사항 확정, DB 스키마 설계, UI/UX 와이어프레임, 개발 환경 구축, API 설계

WEEK 3-4

핵심 백엔드 개발

관리자 인증, 고객 데이터 CRUD, KASI API 연동, 사주 데이터 정규화 엔진 구현

WEEK 5-6

AI 엔진 & 에디터

LLM 프롬프트 엔지니어링, AI 초안 생성 파이프라인, WYSIWYG 웹 에디터 개발

WEEK 7-8

PDF 엔진 & 통합 테스트

PDF 템플릿 디자인/개발, 100+페이지 안정성 테스트, End-to-End 통합 검증

WEEK 9-10

고도화 & 배포

대시보드, 배치 처리, 피드백 반영, 운영 매뉴얼 작성, 최종 배포 및 인수인계
