## 2. 왜 톡톡인가?

### 기존 채널 vs 톡톡

| 채널 | 건당 비용 | 도달률 | 열람률 | 차단 리스크 |
|------|----------|--------|--------|------------|
| 문자 SMS | 20원 | 95% | 15~20% | 높음 (스팸신고) |
| 카카오 알림톡 | 8~15원 | 90% | 30~40% | 중간 |
| 이메일 | 1~5원 | 70% | 10~15% | 낮음 |
| **네이버 톡톡** | **0원** | **98%** | **60~80%** | **낮음** |

### 톡톡 강점
- **비용 0원** — 네이버 톡톡은 무료 메신저
- **열람률 60~80%** — 사업자가 고객 문의로 인식, 거의 다 열어봄
- **신뢰도 높음** — 네이버 플레이스 내 공식 채팅 채널
- **즉시 응답 가능** — 관심 있는 사업자가 바로 답장

## 3. 우리만의 기술적 강점

### 🔍 자체 크롤링 시스템
```
GraphQL API 기반 실시간 수집
├── 네이버 플레이스 전체 업종 크롤링
├── 톡톡 활성 여부 100% API 검증 (OCR 아님)
├── 전화번호 / 주소 / 업종 자동 분류
└── 일 10,000건+ 신규 수집 가능
```

### 🛡️ 스팸 우회 로직
**톡톡 발송:**
- Human-like 발송 패턴 (랜덤 딜레이 15~45초)
- 계정 로테이션 (다중 네이버 계정)
- 시간대별 발송량 조절 (업무시간 집중)
- 메시지 템플릿 변형 (동일 문구 반복 방지)
- 브라우저 핑거프린트 우회 (Canvas/WebGL/UserAgent 랜덤화)
- 마우스 움직임 시뮬레이션 (봇 탐지 우회)

**이메일 발송:**
- SPF/DKIM/DMARC 인증 완료 도메인
- 발송 속도 조절 (ISP별 레이트 리밋 준수)
- 바운스/스팸신고 실시간 모니터링
- A/B 테스트 (제목/본문 최적화)
- 수신거부 자동 처리

### 📊 실시간 모니터링 대시보드
- 발송 현황 (성공/실패/대기)
- 열람률 / 응답률 추적
- 업종별 / 지역별 성과 분석
- 캠페인별 ROI 리포트

## 5. 발송 프로세스

```
① 타겟 설정
   └── 업종 / 지역 / 키워드 선택

② 메시지 작성
   └── 톡톡용 + 이메일용 각각 작성
   └── A/B 테스트 변형 자동 생성

③ 발송 실행
   └── 톡톡: 업무시간 자동 발송 (10시~18시)
   └── 이메일: 최적 시간대 자동 발송 (화~목 오전)

④ 모니터링
   └── 실시간 대시보드 확인
   └── 응답 알림 즉시 전달

⑤ 후속 관리
   └── 응답 업체 → CRM 자동 등록
   └── 미응답 → 3일 후 팔로업 발송
```
