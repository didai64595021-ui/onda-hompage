노출 캡쳐 사이트

서비스 개요

현재 인스타그램 검색 결과는 계정 상태, 사용자의 접속 기기, 접속 시점에 따라 모두 다르게 표시됩니다...

이러한 불편함을 해소하기 위해, 지정한 키워드에 대해 상위에 노출되었을 때 자동으로 화면을 캡쳐해주는 "상위 노출 캡쳐 슬롯"을 도입하였습니다.

노출 캡쳐 슬롯은 광고주 계정 또는 리그램 계정에서 업로드된 콘텐츠가 추천탭 상단 노출된 경우...

주요 기능

자동 캡쳐 다수의 계정 확인 가능 (최대 5개) 최대 20개 키워드 확인 가능 전용 확인 페이지 제공
