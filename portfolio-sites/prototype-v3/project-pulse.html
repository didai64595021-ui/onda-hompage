Challenge

전환율의 벽을 넘어서

PULSE의 기존 이커머스 플랫폼은 전환율이 업계 평균 이하였습니다. 구매 여정의 마찰을 제거하고 매출을 극대화할 수 있는 플랫폼 리디자인이 필요했습니다.

Process

데이터 기반 최적화 프로세스

01

데이터 분석

2주

히트맵, 퍼널 분석, 세션 리플레이를 통해 전환 저해 요인을 정밀하게 파악했습니다.

02

UX 최적화

4주

구매 여정을 간소화하고 결제 프로세스의 마찰 포인트를 제거했습니다.

03

비주얼 디자인

4주

제품 중심의 비주얼 시스템을 구축하고 모바일 퍼스트로 전체 UI를 재설계했습니다.

04

A/B 테스트

3주

주요 페이지별 A/B 테스트를 통해 최적의 디자인 변형을 검증하고 확정했습니다.

Solution

마찰 없는 구매 경험

히트맵 분석 기반 레이아웃 재구성, 1-클릭 결제 도입, 개인화 추천 엔진 연동. 모바일 퍼스트로 전체 리디자인.
